# 2026届上海市静安区高三一模语文·阅读部分

> 说明：以下为题目结构、篇目与参考答案要点整理，题号分值以官方原卷为准（不同来源题号可能相差1）。原卷为"静安区2025学年第一学期教学质量调研 高三语文试卷"（2025.12）。

## （一）现代文阅读一（16分）——甲乙两文（语用学 · 格莱斯合作原则）
【甲】汪锋《您这话是什么意思？》（《读书》2023.1）；【乙】[英]乔治·尤尔《语言研究》（商务印书馆，2023）。围绕语用学中格莱斯"合作原则"的遵守与违反展开。

- **第3题**（3分）分析两文"违反合作原则"的效果，选不正确的一项 → **D**
- **第4题**（3分）判断哪项不属于"故意违反合作原则"的前提 → **C**
- **第5题**（3分）结合甲文第③段，分析画线句的逻辑矛盾与认知统一
  要点：故意称暴行为"伟绩"构成矛盾；预设与国人共享的价值判断；读者识别荒谬后激发否定情感、达成共鸣。
- **第6题**（3分）填空，对比两文的论述方式
  要点：理解言外之意需"基于双方共同认知与默契"；乙文侧重理论分析，以"三明治"含义解析诠释规则。
- **第7题**（4分）分析甲文的论述思路
  要点：由个人经历引入"切口"现象 → 引入合作原则理论 → 以鲁迅反语等案例分析违反的条件与效果 → 指出"语境"对理解言外之意的关键作用。

## （二）现代文阅读二 / 文学类文本（17分）——孙继泉《美丽的闪电》
选自《微型小说月报》2025.1，以儿童视角写"闪电"与成长。

- **第8题**（4分）分析标题"美丽的闪电"的含义
  要点：自然生命力的象征；形态多变、自由奔放精神的体现；短暂易逝的童年的象征。
- **第9题**（4分）从句式角度赏析画线部分
  要点：长短句交织；整散结合（"有的像……"排比）。
- **第10题**（4分）分析第⑦⑧段如何塑造乐乐的形象
  要点：好奇心（冒险看闪电）；有主见、有勇气；富有同情心（"受灾了"）；渴望成长（"无奈"）。
- **第11题**（5分）结合三处加点词赏析结尾的妙处
  要点：儿童与成人在信息、价值观上的隔绝；"不知道"是获得"满足"的前提；走向"村庄"象征现实归宿与终将长大。

## （三）诗歌鉴赏（7分）——《别匡山》（旧题李白）
- **第12题**（3分）对诗歌解说不正确的一项 → **B**
- **第13题**（4分）探讨此诗作者是否为李白，任选角度作答
  要点（认为"不是"）：不押韵；对仗不工；尾句直白俗气；李白极少写七律。
  要点（认为"是"）：符合宽对；按唐代韵部可算押韵；末句昂扬符合李白才情；其早期有律诗名作《渡荆门送别》。

## （四）文言文阅读一（19分）——甲乙两文（褚遂良）
【甲】《旧唐书·褚遂良传》[五代]刘昫、《新唐书·褚遂良传》[宋]欧阳修；【乙】清·赵翼《廿二史札记》。

- **第14题**（2分）加点词释义：(1) 不豫 → **C**；(2) → **D**
- **第15题**（2分）加点词意思：(1) 坚持、坚决；(2) 调任
- **第16题**（5分）翻译"陛下亲承德音……恐不可废"
  参考：陛下亲自聆听（先帝）遗命，话音仿佛还在耳边；皇后从那时起未闻有过错，恐怕不能废黜。
- **第17题**（3分）断句
  参考：此乃陛下家事／不合问外人／帝乃立昭仪为皇后／左迁遂良潭州都督
- **第18题**（2分）分析"还笏""叩头流血"对塑造形象的作用
  要点：表现对皇权的失望与履行托付之责，刻画忠诚而痛苦的形象；以极端方式展现以死进谏的忠烈刚直与悲愤抗争。
- **第19题**（5分）(1) 填表格（2分）要点：主动分析无忌、李勣身份特殊不宜先谏，自己承担进谏之责。(2)（3分）要点：属第一种（当日政要之事）；"扑杀""獠"刻画武则天强悍果决，暗示其已干预朝政。

## （五）文言文阅读二（11分）——袁中道《游岳阳楼记》
- **第20题**（2分）分析不当的一项 → **B**
- **第21题**（4分）分析文中"悲"的内涵
  要点：面对雄浑风浪，个体渺小，生宇宙永恒与人生须臾之悲；景象越"雄快"，越反衬生命萎顿、不能建功立业之悲。
- **第22题**（5分）探究作者表现迁谪主题的独到之处
  要点：滕子京哭立功被诬，袁中道哭壮志消沉、骨肉离世之孤独；本文由传统仕途失意转向对生命流逝、个体孤独、生存意义的追问，拓展了迁谪主题。
