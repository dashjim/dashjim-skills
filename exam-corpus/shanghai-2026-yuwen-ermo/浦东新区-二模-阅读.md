## （一）现代文阅读一（16分）——论述类文本·小说文体演变

文本探讨小说文体的核心特征、演变与现代性挑战。

- **第3题**（2分）解释"压扁"含义 → **A**

- **第4题**（2分）【甲】判断不能证明论断的一项 / 【乙】选支撑观点的论据 → **D**

- **第5题**（3分）对第②—⑤段理解符合文意的一项 → **A**

- **第6题**（4分）概括行文思路  
  **要点**：
  - 先由文体特征切入话题
  - 界定概念引出"虚构"核心特质并论证
  - 探讨"情节与人物"关系需平衡
  - 指出"现代性"挑战需革新回应时代
  - 升华得出"小说是重塑世界的伟大文体"

- **第7题**（5分）反驳读者留言  
  **要点**：
  - 表态需用第二人称回应
  - "重塑世界"指重塑看待世界的方式，非直接改变现实
  - 《老人与海》影响近百年，无法推出"不能重塑世界"结论

## （二）现代文阅读二 / 文学类文本（16分）——《托塔天王》

叙写父子亲情与角色转换。

- **第8题**（3分）赏析第③段比喻画线句  
  **要点**：
  - "胶片""光"紧扣父亲职业身份，比喻新颖
  - 写儿时相伴温馨
  - 暗含父亲带来温暖与陪伴

- **第9题**（4分）分析两次"这是有生以来头一次"作用  
  **要点**：
  - 引出回忆含感恩
  - "最后一次"拉回现实显心疼
  - 首尾呼应显生命流转、亲情延续
  - 对照中浓缩情感变化

- **第10题**（4分）【甲】"轻松口吻写沉重内容"分析 / 【乙】父亲观影细节赏析  
  **要点**：见原卷详细设问（各4分）

- **第11题**（5分）【甲】"托塔天王"称呼构思分析  
  **要点**：
  - 贯穿全文作线索
  - 形象从守护者到被守护者转变
  - 情感层层递进
  - 结尾深化主旨  
  / 【乙】不同叙述视角赏析（童年/成年视角）

## （三）文言文阅读一（19分）——王猷定《樗叟传》（清）

记樗叟侠义、技艺与品德。

- **第12题**（2分）词语解释  
  （1）坐以法 → "定罪判罪"  
  （2）当上流之冲 → "交通要道"

- **第13题**（2分）词义辨析  
  （1）榷 → **A**  
  （2）粥粥 → **D**

- **第14题**（1分）方框处填词 → **C**

- **第15题**（4分）断句  
  **答案**：叟阳与元荫交/而阴趣御史归里/夜乘一驴/送之崇文门外/平时宾客无一人至者。

- **第16题**（5分）翻译  
  **要点**：事迹未详尽记载，只记述他周济他人危难、不畏权势之事，他遭大难得生还是有原因的。

- **第17题**（5分）分析"有道"评价用意  
  **要点**：
  - 技艺之道（医、画）
  - 品德之道（不慕名、侠义、重情、孝友、家风传承）
  - 表达高度肯定，寄托乱世坚守正道的期盼

## （四）诗歌与文言文阅读二（19分）——王鏊《题壑舟图》《壑舟记》（明）

题画诗与记文，探讨"藏舟于壑"的人生态度。

- **第18题**（3分）颈联理解赏析错误项 → **D**

- **第19题**（4分）结合"不脱不黏"分析  
  **要点**：
  - "不脱"于画（颈联描摹实景）
  - "不黏"于画（首、颔、尾联抒情入理）
  - 脱黏相生

- **第20题**（3分）第②段理解正确项 → **B**

- **第21题**（4分）赏析画线句  
  **要点**：
  - 句式长短结合、节奏变化（四字短句转长句）
  - 比喻形象、对比鲜明（"一叶"与"波涛如山"）

- **第22题**（5分）  
  【甲】结合两文评析"兄之见远矣"  
  【乙】谈"藏舟于壑"人生态度的理解与评价（含理解2分+评价3分，可认同或不认同）
