# 2024届上海市松江区高三一模（2023学年上学期期末质量监控）语文·阅读部分

> 来源：https://zy.21cnjy.com/18199057 （二一课件通 / 21世纪教育网，含答案）
> 说明：以下为题目结构、篇目与参考答案要点整理。客观题答案用加粗字母，主观题只记参考答案要点。题号在不同来源可能相差 1，以松江区官方原卷为准。

## （一）现代文阅读一（16分）——科普作品节选（无独立篇名，关于"基因·先天与后天"的科普书节选及读者评论）
文本为一部探讨行为由先天基因与后天环境共同作用的科普书节选，附读者评论情境。

- **第3题**（2分）指出试读内容所在小节标题、可延伸阅读章节
  要点：学习是先天与后天共同作用；（延伸阅读）第八章
- **第4题**（3分）找出与"断言"逻辑谬误不同的一项 → **D**（选 C 给 1 分）
- **第5题**（3分）不符合文意的一项 → **B**
- **第6题**（3分）作者不可能认同的一项 → **D**
- **第7题**（5分）写一则读者评论反驳"小松"的观点
  要点：作者"驳中有立"，既反驳错误观点又阐述"行为由基因和后天共同造就"，故对后天因素的论述是必要的

## （二）现代文阅读二 / 文学类文本（16分）——李新文《条凳》
散文（有删改），以"条凳"为线索，回忆乡土生活与两代女性，思考传统伦理。

- **第8题**（3分）赏析第②段景物描写
  要点：以阳光、溪水、虾米等意象营造明媚灵动的画面，表现"我"对自由生活的向往
- **第9题**（4分）赏析第③段语言风格前后变化
  要点：前段口语化、生动活泼，后段凝练书面语，表达深沉思考
- **第10题**（4分）分析从"祖母"到"新媳妇"的构思作用
  要点：两代女性形成对比，呼应"条凳"的象征意义，展现对传统伦理的思考
- **第11题**（5分）评析第⑧段"回味"与"惆怅"的意蕴
  要点："回味"含自我觉醒与对自由的向往；"惆怅"含对父辈叹息及被束缚的无奈

## （三）诗歌鉴赏（8分）——高适（唐）《陪窦侍御泛灵云池》
- **第12题**（1分）选择与诗风匹配的背景图 → **D**（苍茫空阔）
- **第13题**（2分）理解分析不正确的一项 → **B**
- **第14题**（5分）从"得体"角度赏析本诗
  要点：借景抒情、用典，并以"弱羽"自比，委婉赞美追随之意，既不谄媚又与自身身份相称

## （四）文言文阅读一（19分）——《南唐书·陈乔传》
- **第15题**（2分）加点词义：(1)荫——祖上恩荫；(2)见——被
- **第16题**（2分）选择释义：(1)耽玩 → **D**；(2)以乔为介 → **C**
- **第17题**（3分）三处断句
  要点：元宗愕然曰／微卿／几落贼人彀中／传位之意遂止（三处，以官方原卷为准）
- **第18题**（5分）翻译画线句
  要点：城将陷，后主拟写降书，令陈乔与（徐）仲寓开门纳军；陈乔归府后将降书扔入雨槽（拒不投降）
- **第19题**（3分）分析陈乔进谏的说服力
  要点：从帝位之重要入手，假设后果，引史实层层加深，逻辑严密而有力
- **第20题**（4分）是否同意作者对陈乔的评论
  要点：观点自定，需结合文意分析其忠君之节及"是否死得其所"

## （五）文言文阅读二（11分）——徐霞客（明）《溯江纪源》
- **第21题**（1分）填空选项 → **B**（岷山）
- **第22题**（2分）不符合作者意思的一项 → **C**
- **第23题**（4分）赏析连用五个"不第此也"的妙处
  要点：层层推进、反复强调"江源在昆仑"，蕴含自信情感，增强说理力度
- **第24题**（4分）结论与今考证不符是否影响其价值
  要点：不影响，应肯定其大胆质疑、求真务实的探索精神
