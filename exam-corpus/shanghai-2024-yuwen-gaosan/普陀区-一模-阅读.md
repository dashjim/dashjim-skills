# 2024届上海市普陀区高三一模语文·阅读部分

> 说明：以下为题目结构、篇目与参考答案要点整理。题号在不同转载来源可能相差 1，以各区官方原卷为准。

## （一）现代文阅读一（16分）——米哈里·希斯赞特米哈伊《创造力：心流与创新心理学》
选自介绍创造力与"心流"理论的心理学著作，围绕创新过程的心理机制展开。

- **第3题**（2分）选出不属于本书探讨的"创造力"的一项 → **B**
- **第4题**（2分）本书应归入文献梳理表的哪一类 → **D**
- **第5题**（2分）最可能获取参考信息的课题 → **A**
- **第6题**（4分）完成原创概念表格
  要点：（1）"差序格局"指以自己为中心、由亲属地缘关系决定差等、可伸缩的次序关系社会格局；（2）"心流"用于阐释创新者创新时的心理状态，回答其战胜风险艰难、找到乐趣的过程与原因。
- **第7题**（6分）分析作者强调创新艰难复杂却不违背积极心理学立场的理由
  要点：全面客观总结创新过程，体现学术科学性、增强可信度；不将创新者视为"超人"，纠正只关注最佳状态、忽视风险与创造链三要素的认识；鼓励读者投身艰难创新实践以自我实现，符合积极心理学"研究人类积极心理品质"的宗旨。

## （二）文学类文本 / 现代文阅读二（16分）——张敏珺《广告屏蔽器》
小说，借人物"修斯"被无孔不入的"病毒营销"困扰、购买广告屏蔽器的经历，反思广告对人精神的异化。

- **第8题**（3分）分析"病毒营销"的特点
  要点：商家无孔不入投放广告、鼓励人们加入推广，如病毒般寻找媒介复制传播、不断蔓延。
- **第9题**（4分）分析画线部分的作用
  要点：通过修斯质疑、崩溃大叫的语言描写呈现屏蔽广告信息，表现广告无处不在及修斯的厌烦，推动情节、为其购买屏蔽器作铺垫。
- **第10题**（4分）赏析前后不同叙述视角的妙处
  要点：前半为修斯限知视角，写病毒营销的烦扰及屏蔽后的清静自在；后半转为全知视角，揭示被拦原因，同事评论与修斯"疯了"形成认知冲突；视角转换增强悬念波澜，凸显广告对人生活与精神的扭曲。
- **第11题**（5分）从"思想启迪性"和"艺术震撼力"评价结尾设计（开放题）
  要点：思想启迪性——补笔具警示意味，揭示人类集体异化的悲剧，启发处理自我需求与现实的关系；艺术震撼力——垃圾桶说明书特写发人深省，末段独立成段、冷峻增强反讽；亦可认为结尾过于直白拖沓、削弱回味（言之成理即可）。

## （三）诗歌鉴赏（8分）——（唐）杜审言《和康五庭芝望月有怀》
望月怀人之作。

- **第12题**（1分）可能收录本诗的选集 → **B**
- **第13题**（3分）选出分析错误的一项 → **B**
- **第14题**（4分）从意境营造角度赏析第三联
  要点：营造清寒寂寥的意境，通过"雾濯清辉""风飘素影"等景物与"苦""寒"的心理，情景交融，寄寓思念远人之情。

## （四）文言文阅读一（19分）——节选自《宋史·余靖传》
记叙北宋谏官余靖刚正谏言之事。

- **第15题**（2分）解释加点词
  要点：(1)突出、超出；(2)指责、批评、讥讽。
- **第16题**（2分）为加点词选择释义 → (1)**B** (2)**C**
- **第17题**（5分）翻译画线句
  要点：张昭评论将领，认为鲁肃鲁莽；汉皇、吴主常听此类诋毁却都重用而不猜疑，难道会损其美德吗？
- **第18题**（3分）断句（三处）
  要点：仁宗欲更天下敝事／增谏官员／使论得失／以靖为右正言。
- **第19题**（2分）用四字概括余靖的品质
  要点：刚强正直／刚正不阿。
- **第20题**（5分）分析余靖谏言的说服力
  要点：立场契合仁宗革除弊政的需求；先陈严重后果，再以吕觉旧例比照，指出仁宗言行矛盾；"前日""今日"对比鲜明；劝君深谋远虑、防小人钻营，层层深入。

## （五）文言文阅读二（11分）——【甲】（明）薛瑄《车窗记》；【乙】（唐）刘禹锡《陋室铭》
两文比较阅读，围绕安贫乐道、立身处世展开。

- **第21题**（1分）填入方框的虚词 → **D**
- **第22题**（3分）赏析"富者"神情动作描写的表达效果
  要点：刻画富人侧目讥笑、掩嘴离去的不屑情态，反衬车窗的简陋及作者自身不谋私利的落魄处境，暗含对褊躁富者的嘲讽。
- **第23题**（3分）选出分析正确的一项 → **B**
- **第24题**（4分）辨析两文在立身处世追求上的侧重
  要点：两文都表现安贫乐道；【甲】文侧重御史不因官位膨胀欲望、清廉自守以求心境开阔；【乙】文侧重文人怀才不遇仍保持清高品格、不与世俗同流。
