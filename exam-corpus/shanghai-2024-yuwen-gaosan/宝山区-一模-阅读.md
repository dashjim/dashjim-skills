# 2024届上海市宝山区高三一模语文·阅读部分

> 来源：https://zy.21cnjy.com/18477208 （上海市宝山区2023-2024学年高三上学期12月一模考试语文试卷·解析版，21世纪教育网）
> 说明：本卷为 2024届（2023学年）宝山区高三一模，考试时间约 2023年12月。以下为题目结构、篇目与参考答案要点整理。题号在不同转载来源中可能相差 1，**以宝山区官方原卷为准**。客观题答案用加粗字母，主观题只记参考答案要点。

## （一）现代文阅读一（16分，第3—7题）——《临床医学人文：困境与出路》
说明文/论述类文本，节选第二部分"叙事医学：医师领衔的临床人文突围"，讨论叙事医学的价值与文学（虚构）在临床人文中的作用。

- **第3题**（2分）根据"摘要"判断第一部分标题 → **C**
- **第4题**（3分）对"叙事医学"理解不正确的一项 → **A**
- **第5题**（3分）第④段画线句分析最恰当的一项 → **A**
- **第6题**（4分）评析第③段对文学（虚构）价值辩护是否充分
  要点：可判为辩护不充分——第一条理由推不出"虚构必然产生实效"，"心灵的抚慰与灵魂被救赎"未必能达成；第二条将虚构与情感灵性简单相提并论，以偏概全；整体理由缺乏充分支撑、层次交叉。（若答"充分"并自圆其说亦可）
- **第7题**（4分）评价叙事医学的人文价值
  要点：重视患者疾苦，给予情感抚慰与心理疏导，提升医生的共情与反思能力；有助于慢性病、绝症时代的治疗；有助于缓解医患冲突、改善医患关系；丰富"医乃仁术"的内涵，提升医者人文素养。

## （二）文学类文本阅读（16分，第8—11题）——茹志鹃《惜花人已去》
散文，追念老舍先生，借"惜花""赏牡丹"寄托对老舍人格与命运的怀念与礼赞。

- **第8题**（4分）赏析第④段对画中黑牡丹的描绘
  要点：以"乌绒"喻花瓣，写其色泽与柔美质地；"玉露未干"衬其鲜嫩润泽；运用通感（化视觉为嗅觉等）表现勃勃生机。
- **第9题**（4分）解析老舍先生踏花而行的寓意
  要点：脚步之轻见其爱花、爱美好生活；懂花惜花之心见其心灵之美；落英铺路象征其以死抗争的悲剧命运，以及自我牺牲、无私奉献的精神。
- **第10题**（4分）第⑨⑩段重复描写赏牡丹是否繁冗
  要点：不繁冗。第⑨段回忆赏花，反衬噩耗之痛；第⑩段噩梦后再现赏花之景，以景结情，寄寓赞美与怀念；反复渲染，深化情感。
- **第11题**（4分）结合《百合花》与本文，评析"文学作品应表现人性"
  要点：赞同。《百合花》写军民的淳朴善良，本文写老舍的温和怡然与自尊刚烈；前者是美好人性的赞歌，后者是美好人性被毁灭的挽歌，二者都具有感染力，印证文学应表现人性。

## （三）诗歌鉴赏（8分，第12—14题）——（南朝梁）何逊《从镇江州与游故别》
送别诗，写夜雨话别、依依难舍之情。

- **第12题**（1分）写作方法判断不正确的一项 → **A（夸饰）**
- **第13题**（3分）理解赏析不正确的一项 → **D**
- **第14题**（4分）以"情词宛转，浅语具深"赏析画线句的语言表现力
  要点：首句写室外景，"夜雨""空阶"渲染幽凉空寂，映衬离人心境；次句写室内景，彻夜话别、灯暗如心境；平常之景经熔铸而寓情于景，传达难舍与感伤之情，语浅而情深。

## （四）文言文阅读一（19分，第15—20题）——《殷都传》
人物传记，写殷都准确预判边情、反对轻率发兵，终因触怒权臣被弹劾罢官。

- **第15题**（2分）加点词释义：（1）工——擅长、精通；（2）讽——含蓄劝告、暗示
- **第16题**（2分）（1）"孔道" → **B（通达）**；（2）"课" → **A（考核）**
- **第17题**（5分）翻译画线句
  参考：殷都认为两股敌人不能彼此利用，不应惹急使其联合；掌权者非常认同，后来两支队伍果然依次解围离开。
- **第18题**（2分）断句正确的一项 → **D**
- **第19题**（4分）分析殷都被罢官的原因
  要点：殷都准确预判火落赤、撦力克不会犯边，反对石星轻率发兵；其洞若观火有损石星颜面，石星羞愧嫉恨，遂借言官弹劾将其罢免。
- **第20题**（4分）续写下联（上联"体恤民情施行仁政"）
  示例："洞察实况呈献良策"（或"明察秋毫""洞察先机"等，须对仗工整、切合殷都事迹）。

## （五）文言文阅读二（11分，第21—23题）——（清）钱大昕《〈罗溪志〉序》
序文，为记载罗店（罗溪）历史文化的方志作序，阐发范翊王编志、热爱先人与故乡之意。

- **第21题**（3分）推断《罗溪志》内容最恰当的一项 → **C（记载罗店历史文化的史书）**
- **第22题**（3分）分析引用《诗经》句与太公之言的作用
  要点：《诗经》句表达对故乡的热爱之情，太公语表达对先人的敬爱并推及相关人物；借以体现范翊王的写作意图，赞其热爱先人、故乡的孝悌之心。
- **第23题**（5分）为《宝山区志》提供改革开放以来的一则资料并说明理由
  示例：地铁一号线及三、七、十八号线，逸仙路等高架构成交通网络；理由：便利往来联系、缓解交通压力、促进经济发展，是现代化建设的标志。
