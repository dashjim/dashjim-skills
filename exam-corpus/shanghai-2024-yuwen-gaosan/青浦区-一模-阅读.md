# 2024届上海市青浦区高三一模语文·阅读部分

> 来源：https://zy.21cnjy.com/18290804
> 说明：以下为题目结构、篇目与参考答案要点整理。题号在不同来源常差 1，以青浦区官方原卷为准。

## （一）现代文阅读一（16分）——陈嘉映《白鸥三十载》
选文出自《白鸥三十载》"真理掌握我们"章节，谈真理与人的关系。

- **第3题**（2分）推断本书内容 → 要点：作者三十多年的经历及感悟
- **第4题**（3分）选符合"以真理为前提"表现的一项 → **C**
- **第5题**（3分）选不符合文意的一项 → **D**
- **第6题**（3分）选作者不可能认同的一项 → **B**
- **第7题**（5分）反驳"没必要写第⑦段"的观点
  要点：宽容与获得真理的方式相通，是对内容的补充；结合目录可见文章具有随感、可超越话题的特点。

## （二）现代文阅读二 / 文学类文本（16分）——林斤澜《台湾姑娘（节选）》
小说节选，写"我"与一位台湾姑娘的相遇与分别。

- **第8题**（4分）分析第②段画线句作用
  要点：写出环境破败荒凉、烘托孤寂心境；为下文相遇作铺垫；折射时代背景。
- **第9题**（4分）从叙述视角赏析叙事特色
  要点：以亲历者"我"的限知视角转述，增强真实性与情感体验。
- **第10题**（4分）赏析末段画线句语言
  要点：细节反复刻画分别的不舍；环境描写显无奈与自责。
- **第11题**（4分）从"怪味豆"角度赏析
  要点：叙事怪（淡化情节）、人名怪、选材特别，却塑造出质朴的姑娘形象、赞美人性、关切两岸。

## （三）诗歌鉴赏（8分）——废名《街头》（1937）、李商隐《乐游原》
两首涉及"寂寞/伤感"意象的诗对读。

- **第12题**（1分）两诗语言的共同特点 → 要点：节奏感 / 音乐美 / 韵律美
- **第13题**（3分）选分析不正确的一项 → **A**
- **第14题**（4分）分析两诗意象抒情的不同
  要点：《街头》以意象群强化寂寞；《乐游原》以"夕阳"抒美好易逝之伤感。

## （四）文言文阅读一（19分）——《宋史·何郯传》（节选）
史传节选，写何郯直言进谏之事。

- **第15题**（2分）释词 → 要点：（1）劳＝功劳；（2）进＝推荐
- **第16题**（2分）选义项 → （1）**A**；（2）**B**
- **第17题**（4分）翻译"君臣交疑……不可得矣"
  要点：君臣相互猜疑，却想要天下没有闭塞之患，是不可能实现的。
- **第18题**（3分）断句 → 张尧佐缘后宫亲 / 叨窃非据 / 外庭窃议 / 谓将处以二府
- **第19题**（4分）比较推理方式的相同处
  要点：均为类比推理（由小推大）。
- **第20题**（4分）分析进谏艺术
  要点：对象意识强、抓住对方心理；针对性强、说理逻辑严密。

## （五）文言文阅读二（11分）——汪琬《送徐原一归昆山序》（清）
赠序，借玉山与玉、玉与君子的关系赞美并期许徐原一。

- **第21题**（3分）赏析第③段画线句的表达效果
  要点：假设、反问表祝愿与信心；排比突出徐原一的才德。
- **第22题**（3分）选分析正确的一项 → **C**
- **第23题**（5分）分析行文思路
  要点：由"玉山不产玉"的质疑，引出玉与君子的关系及"寓意于君子"的猜想，为写徐原一的君子表现作铺垫，末以"玉山之得名者，果不虚也"作赞美与期待。
