# 2024届上海市长宁区高三二模语文·阅读部分

> 来源：https://zy.21cnjy.com/19831443 （原卷版+解析版）、https://zy.21cnjy.com/19790241 （含答案），两处内容相互印证。
> 说明：以下为题目结构、篇目与参考答案要点整理。题号在不同来源可能相差 1，**以各区官方原卷为准**。阅读部分总计约 70 分。

## （一）现代文阅读一（16 分）——庸安意《跟曹雪芹学园林建筑》、陈从周《园林清话》
三则材料围绕《红楼梦》大观园与"虚拟园林"的营造展开（材料一、二出自《跟曹雪芹学园林建筑》，材料三出自陈从周《园林清话》）。

- **第3题**（2分）解释材料三中"捉弄"的含义
  要点：指虚实（真假）相生、引人遐思的造园手法。
- **第4题**（3分）对三则材料"引用"的分析，选不正确的一项 → **B**
- **第5题**（3分）选能支持"（园林）首先为文学服务"这一观点的一项 → **B**
- **第6题**（3分）对宝玉评价稻香村一段话的分析，选最合适的一项 → **D**（选 C 得 1 分）
- **第7题**（5分）替"小宁"写一段论坛提问，引发网友讨论现实中建造大观园的不足
  要点：以提问形式，兼顾建造成本、艺术/传世价值、现实限制、审美空间等角度。

## （二）现代文阅读二 / 文学类文本（16 分）——老舍《记懒人》
选自 1933 年《益世报》。

- **第8题**（4分）赏析文中反复说"大概是"的表达效果
  要点：强调不确定语气，表现独特的交往方式与思念、揣测的心理。
- **第9题**（4分）分析"海棠"在叙事中的作用
  要点：线索作用、环境（氛围）作用、象征作用。
- **第10题**（4分）分析"我"对情节发展的影响
  要点：既是叙述者又是参与者，形成限制视角，推动情节、蓄势。
- **第11题**（4分）"懒人"形象是否用于讽刺批判人性之"懒"，谈看法
  要点：开放题，需明确看法并给出理由。

## （三）诗歌鉴赏（8 分）——（宋）万俟咏《长相思（二首）·雨》《长相思·山驿》
两首《长相思》对比阅读。

- **第12题**（3分）对两首词的比较，选最不适切的一项 → **D**（选 C 得 1 分）
- **第13题**（5分）赏析"愁人不喜听"与"行人不要听"两句抒情效果的相似之处
  要点：均写主观愿望——不想听却不得不听，反衬、深化寂寞孤独之情。

## （四）文言文阅读一（19 分）——《薛端传》（《北史》卷三十六）
- **第14题**（2分）加点词释义：（1）据：占据；（2）拔：回转/调转（退兵）。
- **第15题**（2分）选择释义：（1）"喻" → **A**（开导）；（2）"康" → **B**（治理）。
- **第16题**（3分）文中画线部分断句（三处）
  参考：端以为三方未一 / 遽正名号 / 示天下以不广 / 请待龛翦僣伪。（此处存疑：断句处数与各来源转述一致，仍以原卷标点为准）
- **第17题**（5分）翻译"自居选曹……未尝升擢之"
  要点：（薛端）自任选官之职以来，选拔贤能，对才劣行薄者不曾加以提拔。
- **第18题**（7分）以周文帝的视角写一段约 200 字的人物短评（白话）
  要点：概述薛端事迹 + 作出评价，叙议结合。

## （五）文言文阅读二（11 分）——（宋）苏洵《木假山记》（《嘉祐集》卷十五）
- **第19题**（1分）填入方框处的虚词 → **A**（而）
- **第20题**（2分）关于"幸/不幸"，选与文意不符的一项 → **B**
- **第21题**（3分）关于写作特点，选最不适切的一项 → **C**（选 D 得 1 分）
- **第22题**（5分）结合全文，赏析楼昉所评"起伏开合，有无限曲折"的妙处
  要点：三段层层推演、回环往复，主旨渐次深化，扣人心弦。
