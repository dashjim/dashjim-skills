## （一）现代文阅读一（16分）——《浅谈中国的社会分层与教育》（社科论文节选）
论述家庭资本差异与教育、社会阶层流动之间的关系。
- **第3题**（3分）选择应加入第②段的引文 → **B**
- **第4题**（2分）填空补全论证 → 要点：不同家庭的子女在继承和积累的（经济/文化/社会）资本上存在差异
- **第5题**（3分）对文章内容的理解判断 → **C**
- **第6题**（3分）为相应部分选择小标题 → **B**
- **第7题**（5分）"小镇做题家"现象能否支持本文观点
  要点：开放题，可正面或反面作答。须紧扣文中"教育与阶层跃升/资本差异"的核心论述展开，言之成理、结合现象分析即可。

## （二）现代文阅读二 / 文学类文本（16分）——傅菲《酸橙》
散文，写作者对酸橙由期待到失望、再到重新认识其价值的过程。
- **第8题**（4分）赏析第①段画线部分的表达效果
  要点：调动视觉、触觉、味觉等多种感官，侧面描写，与后文形成反差，为下文情感转变作铺垫。
- **第9题**（4分）赏析第⑧⑨段人物语言的特点
  要点：多用口语、短句，质朴自然，就近取譬，符合人物身份与乡土语境。
- **第10题**（4分）梳理作者对酸橙认识的变化
  要点：期待 → 失望 → 发现其独特价值 → 领悟"无用于此者或有用于彼"（换一处境即有用）。
- **第11题**（4分）用酸橙比喻《红楼梦》中的人物
  要点：开放题，选贾宝玉（或贾琏、王熙凤等）皆可，须结合酸橙特点与人物性格自圆其说。

## （三）诗歌鉴赏（8分）——【宋】俞紫芝《咏草》
咏物诗，多角度写草，寄寓人世沧桑之感。
- **第12题**（1分）本诗可能被收录的选集 → **B**
- **第13题**（2分）本诗写作特点的判断 → **C**（托物/咏物抒情）
- **第14题**（5分）赏析前六句从多角度咏草
  要点：由近及远、由实到虚，拓展时空，今昔对比，借草的荣枯抒发人世沧桑、兴亡之感。

## （四）文言文阅读一（18分）——《后汉书·荀彧传》（节选）
记荀彧择主、辅曹一段史事。
- **第15题**（2分）解释加点词 → 要点：（1）暗中；（2）机会（时机）
- **第16题**（2分）加点词释义选择 → （1）**C**；（2）**D**
- **第17题**（3分）断句 → "时曹操在东郡／彧闻操有雄略／而度绍终不能定大业／乃去绍从操"
- **第18题**（7分）句子翻译（含张邈遣人诳荀彧一段）
  要点：忠实直译，落实"诳/度/终"等关键实词与句意，文从字顺。（具体译文以官方原卷参考答案为准，此处存疑）
- **第19题**（4分）分析文中矛盾冲突对人物形象的塑造
  要点：借荀彧与曹操（及时局）的冲突，凸显荀彧忠于汉室、坚守操守，反衬曹操雄略而心机深沉。

## （五）文言文阅读二（12分）——【元】李孝光《大龙湫记》
游记，写大龙湫瀑布之雄奇。
- **第20题**（3分）赏析"捣"字之妙
  要点：以拟人/比拟手法，化静为动，写出瀑布倾泻磅礴、势不可挡的气势。
- **第21题**（3分）相关分析判断的选择 → **C**
- **第22题**（6分）分析写游人举动、感受的作用
  要点：以游人反应侧面烘托水声之巨、瀑布之壮，反衬其雄伟，渲染现场氛围。
