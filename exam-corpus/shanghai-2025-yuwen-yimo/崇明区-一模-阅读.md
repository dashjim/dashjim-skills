# 2025届上海市崇明区高三一模语文·阅读部分

> 来源：https://zy.21cnjy.com/21970367 （21世纪教育网，2025届第一次模拟考试，含答案）
> 说明：以下为题目结构、篇目与参考答案要点整理。以各区官方原卷为准。

## （一）现代文阅读一（17分）——甘露《虚构与现实，边界从未消失》（《事实与虚构——论边界》书评）
- **第3题**（2分）"泛虚构主义"概念理解 → **A**
- **第4题**（3分）关于阅读"预期"不适合的一项 → **C**
- **第5题**（3分）依据本文可作的推断 → **D**
- **第6题**（4分）评析博物馆虚构故事活动现象
  要点：态度1分；"消解历史物品价值、造成历史误解"1点；"消解虚构与事实边界、形成泛虚构主义"1点。
- **第7题**（5分）拟写推荐提纲
  要点：研究对象、理论支点、写作路径、阅读意义，各2分。

## （二）现代文阅读二 / 文学类文本（15分）——胡性能《张掖的时光书》
- **第8题**（4分）结合达利《记忆的永恒》补写第④段两处画线内容
  要点：画面内容概述2分；画面隐含寓意（时间转停、自然永恒与事物变形）2分。
- **第9题**（4分）比较第⑦⑨段画线部分的语言表现力
  要点：语言形式2点、表述效果2点、比较视角1点。
- **第10题**（3分）分析标题"张掖的时光书"的内涵
  要点：外形之书（山如指针）、内涵之书（时光快慢、沧海桑田）、见证之书（人生短促与时间永恒）。（原卷此处题号标注有缺，以官方原卷为准）
- **第11题**（4分）从构思角度赏析
  要点：前半设想蓄势、后半所见反差、内容作用效果。

## （三）诗歌鉴赏（8分）——【甲】《缚鸡行》、【乙】《东西船行》
- **第12题**（1分）可收录两诗的选本 → **A**
- **第13题**（2分）写作特点说法不正确的一项 → **B**
- **第14题**（5分）甲诗结尾是否妙于乙诗
  要点：认同或不认同均可。共同结构1分、事理概括2分、结尾效果比较2分。

## （四）文言文阅读一（18分）——节选自《汉书》（朱云传）
- **第15题**（2分）加粗词释义：(1) (2)
  要点：(1)勉励/振奋 (2)表彰。
- **第16题**（2分）加粗词选择释义：(1) → **B**；(2) → **D**
- **第17题**（1分）填入方框处 → **A**（乃）
- **第18题**（5分）断句（五处停顿）
  要点：元帝好之／欲考其异同／令充宗与诸《易》家论／充宗乘贵辩口／诸儒莫能与抗／皆称疾不敢会。（断处以官方原卷为准）
- **第19题**（4分）翻译画线句
  要点：每分句1点，共译"匡主""益民""尸位素餐""鄙夫"等句。
- **第20题**（4分）结合材料理解"狂狷"
  要点：文本内容概括3分；"狂狷"内涵（直道而行、刚正不屈、有所不为）2分。

## （五）文言文阅读二（12分）——叶燮《密游集序》
- **第21题**（3分）填表比较"才人之诗"与"志士之诗"
  要点：(1)位高权重也有不传 (2)志之所至不得不作 (3)诗歌因诗人而传。
- **第22题**（2分）对第①段分析不正确的一项 → **A**
- **第23题**（3分）分析第②段画线部分的论证作用
  要点：论据典型性1分；论证角度及作用2分。
- **第24题**（4分）赏析为诗集作序的写法
  要点：前后内容概括2分；表达意图分析2分。

> 注：作文题（70分）不属阅读部分。本卷诗歌鉴赏含12—14题、文言文一为15—20题、文言文二为21—24题，题号编排与部分区不同，以官方原卷为准。
