## （一）现代文阅读一（17分）——论述类·关于《红楼梦》大观园

**篇目/材料**
- 材料一：戴志昂《谈〈红楼梦〉大观园花园》
- 材料二：顾平旦《从艺术的“大观”到现实的“萃锦”》

- **第3题**（2分，选择）→ 对“大观园”内涵理解不同的一项：**D**
- **第4题**（3分，选择）→ 不能印证作者观点的一项：**B**
- **第5题**（3分，选择）→ 依据文意可以推断的一项：**B**
- **第6题**（4分，主观）→ 为文章选配图片（图一/图二）并阐释理由。
  - 可选图一或图二（判断本身不作评分依据），依论述合理程度给分。
  - 选图一：能直观呈现恭王府花园景色位置与命名，吸引读者与原著比对。
  - 选图二：能印证大观园园景繁多恢宏，激发读者想象。
  - 评分：文本内容概括1分 + 画面内容描述1分 + 理由及效果2分。
- **第7题**（5分，主观）→ 拟写读书报告会讲稿片段（比较两则材料）。
  - 材料一重在反驳旧红学家观点，认为大观园是曹雪芹基于生活的创造，启示“文学形象未必有实物原型”。
  - 材料二重在探究误会成因，推断恭亲王仿“大观园”改建府园，启示可从园林设计角度欣赏萃锦园。
  - 评分：不同角度1分；材料一观点1分；材料二观点1分；两材料启示各1分；语言特点酌情±1分。

## （二）现代文阅读二（16分）——文学类文本·散文

**篇目**：田鑫《牧羊人》

- **第8题**（3分）→ 赏析第②段的外貌描写。
  - 写皮袄与两腮之黑，表现长久户外劳作特点；勾勒眼神凸显专注敏锐；简洁传神，表现人与环境相融的形象。
  - 评分：色彩分析1分 + 神态分析1分 + 形象特点1分。
- **第9题**（3分）→ 分析引用《暮色》一诗的用意。
  - 描绘夕阳暮归的美好情景，增添诗意；从“曙光”到“晚星”体现放牧生活日日重复；三个“带回”反复吟咏，勾连童年，表现放牧生活温馨美好、始终如一。
  - 评分：诗意效果1分 + 内容分析2分。
- **第10题**（5分）→ 从构思角度分析文章“写羊”的用意。
  - 王学仁坚守土地，其羊训练有素、保有本性；老赵受现代文明影响，其羊邋遢、缺乏专注；外乡人（高汉新）与土地疏离或盲目依赖科技，其羊呆滞木讷、丧失本性。
  - 牧羊人与羊的生活状态高度吻合，借羊表现现代化冲击下的不同生活状态及对乡土疏离的反思，隐含对自身生存状况的担忧，使哲思形象可感。
  - 评分：羊的特点1分 + 人羊关系1分 + 对乡土反思2分 + 对自身担忧1分 + 效果1分，给满5分为止。
- **第11题**（5分）→ 评析以“交响歌剧”为喻是否削弱了主题。
  - 可持“削弱”或“未削弱”（判断本身不作评分依据）。
  - 未削弱：以“交响歌剧”为喻，展现不同生活方式在同一土地共存相生的和谐，是征拆前一曲悲壮挽歌，增强宏大叙事感。
  - 削弱：西方艺术样式与烟村墩质朴风格反差大，一扫原本哀愁反思，破坏意境。
  - 评分：前文内容概括1分 + 主题解读2分 + “交响歌剧”喻的分析2分。

## （三）诗歌鉴赏（7分）

**篇目**：〔南朝〕鲍照《赠傅都曹别》

- **第12题**（2分，选择）→ 对本诗评价恰当的一项：**C**
- **第13题**（5分，主观）→ 将本诗归入“赠别情深”或“咏物抒怀”并阐释。
  - 两种归类均可（判断本身不作评分依据）。
  - 以轻鸿喻傅都曹、孤雁自指；相遇时惺惺相惜的深情；离别的留恋怀念；离别后的孤独寂寞；或以“孤雁”寄托对现实处境的无奈；托物抒怀的效果。
  - 评分：六个要点各1分，给满5分为止。

## （四）文言文阅读一（19分）

**篇目**：王安石《尚书兵部员外郎知制诰谢公行状》

- **第14题**（2分）→ 加点词释义：（1）赞许/推崇　（2）直谏/规劝
- **第15题**（2分）→ 选择释义：（1）**B**　（2）**B**
- **第16题**（4分）→ 翻译第②段画线句。
  - 参考译文：他每到一处都会大力兴建学校。为庄懿、明肃太后在河南建造两座陵墓时，没有向百姓索取任何物品就完成了，这都是谢绛的功劳。
  - 评分：省略主语、“兴”、“起二陵于河南”、“于民”、判断句，一点1分，扣完为止。
- **第17题**（4分）→ 断句（四处）。
  - 及知制诰／自以其近臣／上一有所不闻／其责今豫我／愈慷慨欲以论谏为己事
- **第18题**（3分）→ 赏析用词之妙（“积/发/立/弛”，可对比欧阳修）。
  - “积”“发”表明案件积沉已久无人揭发，见谢公魄力；“立”见果断；“弛”见对从犯的宽厚。塑造出处事有分寸、能力出众的官员形象。
  - 评分：“积”“发”1分 + “立”1分 + “弛”1分 + 官员形象综合1分，给满为止。
- **第19题**（4分）→ 结合事例分析欧阳修对谢公的评价。
  - 欧阳修赞谢公德行贤明、仁义、尽职。面对君王贵戚失德失位能直言进谏、驱赶方士、提议规范诏令流程，见其明辨是非、坚持正道；为同僚奔走、多次为百姓上书、清廉为官、死后家无余财，见其品德高尚、富有仁爱。
  - 评分：欧阳修评价之意1分 + 结合事例分析3分。

## （五）文言文阅读二（11分）

**篇目**：甲 〔明〕归有光《守耕说》；乙 〔明〕王宠《守耕记》

- **第20题**（2分，选择）→ 对甲文第②段内容分析正确的一项：**D**
- **第21题**（1分，选择）→ 填入乙文方框处：**C**
- **第22题**（3分）→ 赏析画线部分的短句。
  - 以简洁明快语言写陈朝用出身豪门、有丰厚田产却超脱世俗、享守耕之乐，强调其能忍受外在诱惑、坚守本心，表现作者对其以务农为本、坚守志向的赞赏推崇。
  - 评分：短句效果1分 + 表情达意2分。
- **第23题**（5分）→ 分析“说”“记”两种文体的写法特色。
  - 甲文（“说”）借居室之名赞沈翁力耕，引发对“君子之耕”与“君子之学”关系的思考，含蓄批评以“不耕”之名行安逸之实的现象，重在议论。
  - 乙文（“记”）借画作题跋描绘守耕隐居场景，赞其坐拥田产仍守本心，引导思考“耕”与“志”的关系，融记叙、描写、抒情、议论于一炉。
  - 评分：“说”“记”写法特色1分 + 两文具体分析各2分。
