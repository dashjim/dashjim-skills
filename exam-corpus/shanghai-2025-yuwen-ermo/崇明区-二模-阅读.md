## （一）现代文阅读一（17分）——访谈：洪晃对谈尤瓦尔·赫拉利（《智人之上》与人工智能主题）
媒体人洪晃与历史学家赫拉利围绕新著《智人之上》，就人工智能与人类未来展开访谈。

- **第3题**（2分）下列对访谈准备与提问技巧的分析，不正确的一项 → **（未提供 / 存疑）**
- **第4题**（3分）对连续使用"AlphaGo"事例作用的分析，不正确的一项 → **（未提供 / 存疑）**
- **第5题**（3分）根据材料可以做出的推断，正确的一项 → **（未提供 / 存疑）**
- **第6题**（4分）为结尾画线处补足访谈内容（100字左右）
  要点：人类的独特优势在于意识与情感体验，AI 无法真正感知；应深化对生命意义的探索、保持思维的主体性、明辨信息真伪。
- **第7题**（5分）设计"人工智能与人类共生"主题展馆的核心展区
  要点：给出展区名称（示例"我的内心你别猜"）+ 互动体验设计 + 设计理念（约150字），需体现人机共生主题与人的主体性。

## （二）现代文阅读二 / 文学类文本（16分）——卞毓方《红土地五彩缤纷》
写红土地的自然景观与生活在其上的人物，借景写人、景人交融。

- **第8题**（4分）从人物描写角度赏析第②③段
  要点：运用肖像/服饰等细节描写；以游客反应侧面衬托；以类比、比喻（《父亲》"原住民""铜塑"）刻画；概括出人物质朴、坚韧的特征。
- **第9题**（3分）分析第⑯段画线句的内涵
  要点：表层指形象的延续并非靠投票选举确立；深层指红土地的文化精神具有自发性与内在力量，文化选择的代表人物具有自主性。
- **第10题**（4分）第25~30段是否可以删去，谈看法
  要点：开放题，可答"不认同（不可删）"或"认同（可删）"，须结合内容与主旨自圆其说；参考倾向为不可删，因其深化人文内涵。
- **第11题**（5分）评析文章以景为题却大量写人的写法
  要点：标题既指自然景观之绚丽，也暗示人文之多元；人物作为载体，实现自然与人文的交融，深化"红土地五彩缤纷"的主旨。

## （三）诗歌鉴赏（7分）——杜甫《夜闻觱篥》
写夜闻觱篥之声引发的漂泊感伤与家国之痛。

- **第12题**（2分）下列对本诗的分析，正确的一项 → **（未提供 / 存疑）**
- **第13题**（5分）赏析尾联"透过一层"的写法
  要点：全诗借意象营造寒夜凄凉之境；尾联以对方（他人）口吻落笔，"知干戈满"与"不见行路难"形成对比，深化漂泊之苦，使家国之痛与身世之悲互相映衬。

## （四）文言文阅读一（19分）——《汉书·史丹传》
记史丹辅佐、劝谏元帝，两度化解太子（后为成帝）储位危机之事。

- **第14题**（2分）解释加点词 → **(1) 被疾：遭受；(2) 希得：少**
- **第15题**（2分）加点词释义选择：(1) 器人；(2) 嘿然而笑 → **（未提供 / 存疑）**
- **第16题**（1分）填入方框处的虚词（且/以/而/因 中选） → **（未提供 / 存疑）**
- **第17题**（4分）画线句断句（断四处）
  参考：臣诚见陛下哀痛中山王／至以感损／向者太子当进见／臣窃戒属毋涕泣／感伤陛下（以官方原卷为准）
- **第18题**（4分）翻译画线句
  参考译文：（史丹）看到定陶王一向很受喜爱宠幸，如今路上流播谣言，为国家起了怀疑之心，认为有太子地位不稳固的议论。
- **第19题**（4分）分析史丹两次化解危机策略的不同
  要点：第②段以乐工作类比、区分音乐技艺与治国才能，"以理服君"；第③段主动揽责、以"保护龙体"作解释，"以情动人"。
- **第20题**（2分）太子最终得以确立（不被废）的原因
  要点：史丹劝谏情理兼备；元帝性格仁慈；太子作为储君的地位已深入人心。

- **第21题**（2分）两文共同强调的学习理念（选择） → **（未提供 / 存疑）**
- **第22题**（4分）从句式角度赏析乙文第②段画线句
  要点：整句概述学习之深入全面，散句收束引发思考；整散结合、节奏感强，增强气势与说服力。
- **第23题**（5分）分析两文的说理特色
  要点：甲文（《学记》）结构严谨、直接说理、系统清晰；乙文（《劝学》）善用对比与比喻论证，生动形象，语言通俗贴近生活。
