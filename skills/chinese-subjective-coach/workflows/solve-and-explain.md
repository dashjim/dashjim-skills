# Workflow: 主观题解题教学 / 打分（判文体 → 定题型 → 按采分点组织）

<required_reading>
**先读通用规律，再按文体读对应模板：**
- 任何主观题都先扫 `references/通用采分规律.md`（八条铁律，尤其"立场影响得分上限"）。
- 论述类（现代文一）→ `references/论述类.md`
- 文学类（现代文二·散文/小说）→ `references/文学类.md`
- 古典诗歌 → `references/古典诗歌.md`
- 文言·记叙（文言文一·传记）→ `references/文言记叙.md`
- 文言·论说（文言文二·论/记/序/书）→ `references/文言论说.md`

拿不准文体时，先读通用规律再判断。每个 reference 内含该文体各题型的"答案必含要素"与自包含范例卡，作答/打分时对照最相近的题型。
</required_reading>

<process>
## Step 1: 确认信息完整
- 检查是否同时有【阅读文本/材料】【题干】（打分任务还需【学生答案】，若有【官方评分说明】更好）。
- 缺原文就停下索要——脱离文本无法判定采分点，评价/分析题官方明写"脱离文本判0-1分"。
- **成功标准**：手上有组织答案（或打分）所需的全部材料。

## Step 2: 判文体
- 看题目属哪个阅读板块：论述类 / 文学类 / 古典诗歌 / 文言记叙 / 文言论说。
- 文体决定"赏析/作用/评价"这些通用动作往哪个方向落（论述落严谨、文学落情感、诗歌落意境、文言落人物或事理）。
- **成功标准**：明确文体，并调出对应 reference。

## Step 3: 定题型，取"答案必含要素"
- 在该文体 reference 内匹配最贴近的二级题型（如"论证思路梳理""段落作用与构思""情感层次""形象概括""评价观点"……）。
- 抄出该题型的"答案必含要素"清单——这就是采分骨架。
- 留意分值与计分方式（"1点1分/点数≈分值"还是"任一点X分/给满为止"还是分档量表）。
- **成功标准**：手上有一张"这道题答案该包含哪几个要素、每个大约几分"的清单。

## Step 4A: 组织答案（作答任务）
- 按采分骨架分块：每个要素写成一个标准小节。
- 每节配**具体文本佐证**（引原文关键词/情节/意象），不空泛。
- 评价题先按"立场影响得分上限"选立场，再表态→分层依据→效果/意义。
- 一句话收束主旨/效果（赏析、构思题的情感/主旨落点几乎必写，漏则扣分）。
- 点到即透，不堆砌；"任一点X分"类多列一个短点保底。
- **成功标准**：答案每个采分点都能独立命中，分点清晰。

## Step 4B: 按采分点打分（打分任务）
- 以官方评分说明（无则用 reference 反推的采分骨架）为唯一尺子。
- 逐个采分点判定学生答案【命中/未命中】——措辞可不同，意思对应即算命中。
- 给出得分/满分，列出命中点与遗漏点。
- **成功标准**：打分完全可回溯到采分点，不凭个人审美加减分。

## Step 5: 讲清"这道题怎么套模板"（教学）
- 用箭头把答案的每个要素对应到采分点，示范这类题的固定套路。
- 一句话点出这类题最值得记住的采分习惯（如"作用题必写主题维""评价题先选高分立场"）。
- **成功标准**：学生下次遇同题型能自己拆采分点。
</process>

<output_format>
建议结构（依题目繁简调整）：

**文体 / 题型**：___ / ___（分值 ___，计分方式：___）

**答案必含要素**（采分骨架）：① ___ ② ___ ③ ___

**规范答案**：
（按要素分点写，每点带文本佐证）

**模板拆解**：答案①→采分点___；②→采分点___；③→采分点___

**一句话心法**：___

——若为打分任务，则用：
**逐点批改**：采分点①【命中/遗漏】…② …｜**得分 X/满分**｜**遗漏/改进**：___
</output_format>

<anti_patterns>
避免：
- 没有原文就直接写答案 / 打分。
- 判错文体（拿论述类的"严谨"去套散文赏析）。
- 只堆手法术语、不结合文本（"运用了对比"却不举原文）。
- 赏析、构思题漏写情感/主旨落点。
- 评价题不选"得分上限更高"的立场，或表态后不结合文本分层。
- 把答案写成小作文——评分只认要素命中，不因辞藻加分。
- 打分时凭语感加减分，不回溯采分点。
</anti_patterns>

<success_criteria>
- [ ] 信息完整性已确认（缺原文则已索要）
- [ ] 文体已判定，调出对应 reference
- [ ] 题型已匹配，取出"答案必含要素"采分骨架
- [ ] 答案分点组织、每点有文本佐证（或打分逐点回溯采分点）
- [ ] 评价题已按"立场影响得分上限"择立场
- [ ] 赏析/构思题写了情感/主旨落点
- [ ] 附模板拆解或一句可迁移心法
</success_criteria>
