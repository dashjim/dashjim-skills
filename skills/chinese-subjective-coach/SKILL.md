<essential_principles>
## 核心心法：把主观题当"采分点清单"来答

选择题的功夫是**排除干扰项**；主观题的功夫是**命中官方采分点**。二者共同内核：答案必须能被官方评分说明一条条勾对上。

1. **先判文体，再定题型**：同一句"赏析画线句"，论述类落"严谨/说服力"、文学类落"生动/情感"、文言落"人物/事理"、诗歌落"意境/情感"，绝不能混用。文体错则整套模板错。
2. **答案 = 采分点分块 + 逐点文本佐证 + 收束**：按该题型"答案必含要素"拆出标准小节，每节配具体文本证据，最后一句点主旨/效果。这是本 skill 经双盲质检验证有效的作答结构（9题42/42命中官方全部采分点）。
3. **分点即分数，宁多勿漏**：多数题"1点1分"或"每点2分封顶"；"任一点X分/给满为止"类，宁可多列一个短点保底，不要在单点上堆砌。点到即透，不写小作文。
4. **⚠️ 立场影响得分上限**（评价题应试关键）：命题常预设某方更优、评分不对称封顶。要选"得分上限更高"的立场——通常是"不可删/不矛盾/不充分/逆常识判断"的一方。
5. **必须有原文**：脱离阅读文本谈答案就是编。缺文本先索要——评价/分析题"脱离文本判0-1分"。
</essential_principles>

<intake>
**先判断信息是否完整：**
- 有【阅读文本/材料】+【题干】→ 进入解题教学 workflow。
- 只有题干、没有原文（现代文赏析、文言分析、诗歌鉴赏都必须有原文才能判定）→ 先索要原文。
- 用户给了【学生答案】要打分 → 同样需要原文+题干+（若有）官方评分说明，按采分点逐条批改。

**然后判文体、读对应 reference。**
</intake>

<reference_index>
所有答题模板在 `references/`，每个文体 reference 内按"二级题型"组织，每题型三块：**考什么/思考动作 → 答案必含要素（从官方评分说明反推的采分点）→ 自包含范例卡（真题原文+官方答案+评分说明+模板拆解）**。

- **文学类.md** — 语言句子赏析/描写作用/段落作用与构思/标题意蕴/叙述视角与人称/形象概括/评价类/文学短评迁移
- **古典诗歌.md** — 炼字/情感层次/手法赏析/结构赏析/两诗比较/评价类
