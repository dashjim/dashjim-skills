<overview>
语言文字运用选择题不依赖长文段，考的是语言规则本身。主要题型：成语（词语）辨析、病句辨析、标点、语句衔接（排序/填句）、语言得体。每种题型有其固定的"错误设置点"，逐一对照即可。
</overview>

<category name="第一大类：成语 / 词语辨析">
题干多为"下列加点成语使用正确/不正确的一项"。

<trap name="1. 望文生义">
- **特征**：只按字面猜意思，忽略成语的实际含义。
- **例子**：把"文不加点"误当"写文章不加标点"（实为"文思敏捷、一气呵成"）；"差强人意"误当"不满意"（实为"大体令人满意"）。
</trap>

<trap name="2. 褒贬误用">
- **特征**：贬义词用于褒扬对象，或反之。
- **例子**："始作俑者""趋之若鹜""处心积虑""无所不为"皆贬义，不能用于赞扬。
</trap>

<trap name="3. 对象/范围用错">
- **特征**：成语有特定适用对象，用错对象。
- **例子**："豆蔻年华"专指十三四岁少女；"汗牛充栋"只形容书多不能形容其他；"美轮美奂"专形容房屋高大众多。
</trap>

<trap name="4. 谦敬错位">
- **特征**：谦辞用于他人、敬辞用于自己。
- **例子**："蓬荜生辉"是谦辞（说自家），不能用于夸别人家；"抛砖引玉"只能自谦。
</trap>

<trap name="5. 语义重复 / 自相矛盾 / 搭配不当">
- **特征**：成语与句中其他成分意思重复或冲突。
- **例子**："忍俊不禁地笑了"（"忍俊不禁"已含笑）；"莘莘学子们"（"莘莘"已表众多）。
</trap>
</category>

<category name="第二大类：病句辨析">
题干多为"下列句子没有语病的一项"。六大类型：

<trap name="6. 语序不当">
- 多层定语/状语顺序错、关联词位置错、逻辑顺序颠倒。
- **破绽点**：主语相同时关联词在主语后，不同时在主语前。
</trap>

<trap name="7. 搭配不当">
- 主谓、动宾、修饰语与中心语搭配不当；一面对两面（"能否…是…的关键"却只答一面）。
- **破绽点**：出现"能否""是否""有没有"要查前后是否两面对一面。
</trap>

<trap name="8. 成分残缺 / 赘余">
- 缺主语（滥用介词"通过…使…"导致主语丢失）、缺宾语中心语；或成分多余重复。
- **破绽点**：句首连用"通过/经过/由于…使/让…"往往主语残缺。
</trap>

<trap name="9. 结构混乱（句式杂糅）">
- 两种句式硬套在一起。
- **例子**："其原因是由…造成的"（"原因是…"与"由…造成的"杂糅）；"本着…为原则"。
</trap>

<trap name="10. 表意不明（歧义）">
- 一句话可作多种理解。
- **例子**："他背着老师和同学把票买了"（"和同学"归属不明）。
</trap>

<trap name="11. 不合逻辑">
- 自相矛盾、概念并列不当、否定失当（多重否定用反）、强加因果。
- **例子**："能否防止这类事故不再发生"（否定叠加致意思相反）；"报刊、杂志、书籍和一切出版物"（概念包含关系并列）。
</trap>
</category>

<category name="第三大类：标点、衔接、得体">

<trap name="12. 标点使用">
- **常见错误点**：概数间不用顿号（"三四十"不加顿号）；并列引号/书名号之间不用顿号；问号——非疑问句用了问号（"我不知道他来不来"是陈述，句末用句号）；冒号管辖范围过大或重复；引号内外点号位置。
</trap>

<trap name="13. 语句衔接（排序 / 填句）">
- **破绽点**：话题一致（主语/陈述对象前后统一）、时间空间逻辑顺序、代词指代对象、关联词呼应、首尾勾连（顶真式重复词）。
- **方法**：抓中心话题定首句，抓代词/关联词/重复词串联，代入验证语意连贯。
</trap>

<trap name="14. 语言得体">
- **破绽点**：谦辞敬辞误用（"令尊/家父""惠赠/敬赠""拙作/大作""光临/前往"）、语体色彩（口语/书面语混用）、场合对象不当（对长辈用命令语气）。
- **例子**：请别人来用"光临"，自己去别处不能用"光临"；称对方父亲用"令尊"，称自己父亲用"家父"。
</trap>
</category>

<ultimate_method>
**语用题破解要点**
- 成语题：望文生义、褒贬、对象、谦敬、重复——五个角度逐一排查。
- 病句题：六大类型（语序、搭配、残缺赘余、杂糅、歧义、不合逻辑）像清单一样过；先找主干（主谓宾）再看枝叶。
- 衔接题：话题一致 + 逻辑顺序 + 代词/关联词呼应，代入验证。
- 得体题：分清谦敬、场合、语体。
- 与阅读题不同，语用题的"原文"就是语言规则本身，规则记牢即可判定。
</ultimate_method>
