通用采分规律 · 八条铁律 + 四条补丁

从 18 份上海各区高三一模/二模原卷的官方评分说明中反复出现、跨文体成立的规律。学生记牢这八条，比记住任何单一模板都管用。作答/打分前先扫这一份，再进具体文体 reference。

八条铁律

  1. 分点即分数，点数≈分值。 几乎所有概括/思路/作用/比较题都是"1点1分"或"每点2分封顶"。作答分点罗列、宁多勿漏，点数略多于分值/2。（浦东2026二模6"1点1分"；徐汇2025一模7"一点1分、两点3分、三点4分"）

  2. 赏析题三段式：手法 + 内容 + 效果 + 情感/主旨。 情感/主旨落点几乎是所有赏析、构思题的固定收束，漏写必扣分。（浦东2026二模8；崇明2026二模8"手法/内容/景物/情感4点满3"）

  3. 作用/构思题三维：内容作用 + 结构作用 + 主题作用。 结构作用要具体（线索/铺垫/呼应/承上启下/推动高潮）。多层面题常有硬约束"只答一层封顶一半分"。（浦东2026二模9/10；嘉定2026二模9"1方面2分、需2层面才满分"）

  4. 评价题三段：表态 + 结合文本分层依据 + 效果/意义。 判断本身多不计分，脱离文本判0-1分。（浦东2025二模7分档量表）

  5. ⚠️ 立场影响得分上限（应试关键）。 命题常预设某一立场更优，评分不对称封顶。考场上要选"得分上限更高"的立场——通常是"不可删/不矛盾/不充分/逆常识判断"的那一方。（奉贤2026二模9"不可删满4、可删最高3"；闵行2025一模22"不充分满3、充分最多2"；崇明2026二模投稿"不录用最高3"；浦东2025一模13"不同意满5、同意最多3"）

  6. 比较题：相同点 + 各文相异点分列定额。 "章法/写法差异"须归因到文体、目的、交际情境、作者身份。（虹口2025一模22"相同1+甲乙各2"；黄浦2025二模23"文体决定章法"）

  7. 迁移/情境写作题双轨给分：内容要素（回扣文本核心概念）+ 文体格式/身份对象意识。 "格式/身份/得体"是最易丢的隐性分，单独占分。（浦东2026一模7"格式1分"；奉贤2026二模7"符合倡议书文体、有呼告语气1分"；青浦2026二模7"第一人称委婉征询"）

  8. 概念/技能题只认精准命中。 下定义须成判断句式（句式本身占分）；词语所指/词义须回填文本语境，泛化字面义零分；断句"错一处扣一分"；翻译"关键实词一点1分"。（嘉定2026二模3"答'看不见的部分'不赋分"；长宁2026二模3"定义表述形式1分"）

四条"易漏易偏"补丁（188 题全量实测归纳）

这四条不是采分结构，而是连高水平作答也系统性栽跟头的盲点——从 188 道上海真题的全量盲做实测中归纳：多数题凭语文功底即可答对，但下面四类失分反复出现、且都能靠一句自检堵上。作答任何赏析/作用/形象/评价/比较题，收笔前过一遍这四条。

补丁一·穷尽两面（形象/意象/象征/评价题最常栽）。 写完一面立刻自问"有没有相反或互补的另一面？"官方答案常把正反两面各设采分点：托物言志的意象既写"孤苦险恶"也要写"高洁劲节"；评价人物既写"退隐仁厚"也要写"勇武担当"；象征义既写"祈福"也要写"生活艰难"的辩证面。只答顺手的那一面，稳丢一半分。（宝山2025二模13乌鹊；杨浦2025二模19；浦东2026一模9滚冰）

补丁二·锁定"题型专属采分轨"，别退回通用套路（母题惯性陷阱）。 题干点明特殊考法时，通用套路（内容主旨、结构过渡、例证多寡、笼统身份）会整轨跑偏： - "评价某文论证/例证是否有力" → 走可比性/典型性/是否以偏概全，不是"例子多不多"。 - "多个人物/古人并置" → 逐个写各自特点再与主体对照，不是笼统归为一类。 - "补记/插叙的作用" → 提炼题干指向的那个性格/主题侧面，不是泛泛"补充交代/结构过渡"。 - "评价小说'讲道理的方式'是否含蓄" → 落叙事如何让道理自然生成（环境铺垫→舒徐节奏→情境化说理→如盐化水），不是对比/象征/标题双关的内容主题解读。（详见 文学类.md 第七节《壶友》范例卡） （宝山2025二模23/9；杨浦2025二模10；徐汇2025一模19）

补丁三·手法/内容答完，必补"效果/趣味/读者作用"层。 这是铁律2/3"情感·主旨落点"里最常被漏的子维度：不仅写"深化主旨"，还要答"这样写达到什么效果、有什么趣味性、对读者有何作用"（乡土名称的童趣、建成的公益意义、激发阅读兴趣）。收笔固定一问："这样写，效果/趣味/读者感受是什么？"（浦东2025二模20景点简介；长宁2025一模10老鸹喝喜酒）

补丁四·立场/排序题，落笔前先校硬约束。 呼应铁律5：开放表态题优先选官方倾向档（"认同"常封顶低一档）；排序/顺序题先验证题干给定的顺序约束再作答，选错方向会连锁丢光该题分。（崇明2025一模6"认同"封顶3；杨浦2025二模6排序）

⚠️ 应试提醒（控篇幅、控时间）

模板范例答案为讲清采分点而写得较充分；实考中，采分点点到即可、每点一句话说透，不必展开成小作文。评分只认"要素是否命中"，不因辞藻加分。分点作答、把要素写全，比把一两点写长更划算——尤其"任一点X分/给满为止"类题，宁可多列一个短点保底，不要在单点上堆砌。

⚠️ 评析论证题的驳斥角度

评析论证/反驳类题只给"某一方不合理"的理由时，优先从逻辑谬误角度切入——以偏概全（"大量"≠"所有"）、偷换概念、转移论题、以特殊代一般，这类角度最贴官方采分口径；凭"操作层面困难/现实可行"等泛泛理由虽可能沾边，但遇到"只判一处、且判错角度"的题会失分。

旧"六型模板"的定位校准

传统答题教学的"六型"（作用/手法/含义/形象/原因/标题）是文学类散文的经典体系，仍有效——但只覆盖【文学类】+ 部分【诗歌】【文言记叙】。上海卷近年增量题型六型覆盖不到，必须新增：论证思路/评析论证（论述类核心）、材料关系分析、推理补全（三段论/顺序）、情境迁移写作（提纲/倡议书/推荐语/劝说）、评价类思辨题（是否/能否/认同）、元评价（论证效果、学术语言是否规范、史传客观性）、跨文本迁移（结合理论卡片/课外文本作答）。

一句话：六型是骨架，本 skill 在其上补齐了"思辨表达 + 信息再加工 + 情境应用"三大新题群——这正是上海卷的拉分/失分焦点。

可靠性：双盲质检背书

本套模板经实战检验：从 5 文体抽 9 题（约占范例卡 30%），"考生 agent"只看文本+题干+模板、不看答案盲做，"独立阅卷 agent"按官方评分说明逐点打分，结果 42/42、得分率 100%——凭模板作答即命中官方全部采分点，证明"答案必含要素"确与官方给分逻辑对齐。

论述类（现代文阅读一）

对应上海卷"现代文阅读一"，文本形态为议论文／科技说明／非连续性文本／复合文本（材料一二三＋链接材料）。本章只收录论述类特有、旧六型（作用／手法／含义／形象／原因／标题）覆盖不到或需要重新校准的新题型。

本章体例：每个二级题型固定写三块——①考什么／思考动作；②答案必含要素（从官方评分说明反推的踩分点）；③自包含范例卡（直接从原卷抄录题干＋依托材料＋官方答案＋评分说明，并做模板拆解，读者无需翻任何别处）。

范例卡取材范围：本轮通读了 4 份原卷的"现代文阅读一"——长宁2026二模（技能交换）、闵行2026二模（道德增强技术）、崇明2026二模（当代文学批评）、浦东2026一模（项飙"透明不透气"）。凡范例卡均出自这 4 卷、逐字照录。要素分析中引用的其他区卷（虹口、徐汇、黄浦等）为既往证据，未在本轮重读，故只作要点性佐证、不做整卡呈现。样本少或本轮未采到真题的题型标注"证据有限"。

一、论证思路 / 行文思路梳理

1. 考什么 / 思考动作 把整篇（或指定段落）拆成"提出—展开—收束"的论证链条，用"首先／接着／然后／最后"给每个逻辑环节贴标签，并点出环节之间的推进关系（层层深入／先破后立／正反对比）。

2. 答案必含要素（踩分骨架：分层切段 ＋ 每层核心动作 ＋ 推进关系／效果） - 要素A＝逐层切分：点数≈分值，一个逻辑环节＝1分。这是本题型最硬的规律。 - 浦东-2026-二模第6题（概括行文思路，4分，文学类范畴）评分说明逐字：切入话题1分／界定概念引出核心特质1分／探讨情节人物关系1分／指出现代性挑战1分／升华主旨1分，明写"1点1分"。 - 虹口-2025-二模第5题（概括论述思路）答案骨架＝提出观点→六大趋势印证→内外因探因→呼吁，四段四点。 - 要素B＝动词化的环节标签：每层要写"提出／界定／阐释／举例／反驳／回应／升华"这类动作词，不能只抄段落大意。 - 闵行-2026-一模第6题（梳理行文思路，4分）＝提出议题→阐释合理性与前提→案例分析→说明适用范围→提出纠正办法；静安-2026-一模第7题（甲文论述思路，4分）＝引"切口"→引合作原则→鲁迅反语案例→指"语境"关键。 - 要素C＝推进关系或论证效果（有则给分）：需点明"层层递进／先破后立／对比"。 - 黄浦-2025-二模第6题（材料二如何层层深入，4分）评分说明"一点1分"，题干已限定"层层深入"。

二、评析论证（合理性 · 充分性 · 针对性 · 论证方法作用）

1. 考什么 / 思考动作 不是复述观点，而是"元评价"——判断论据／反驳／论证是否站得住：论据够不够典型充分、反驳有没有驳到点上、有没有偷换概念或转移论题，然后表态＋分层说理。近年高频变体：多份材料"哪个质疑更合理"、投稿摘要"是否录用"、给定链接材料"能否论证画线句观点"。

⚠️ 质检优化（驳斥角度）：本类题只给"某一方不合理"的理由时，优先从逻辑谬误角度切入——以偏概全（"大量"≠"所有"）、偷换概念、转移论题、以特殊代一般，这类角度最贴官方采分口径。凭"操作层面困难/现实可行"等泛泛理由虽也可能沾边，但遇到"只判一处、且判错角度"的题会失分。（此条来自模板双盲质检：第1题因分值封顶未失分，但驳论角度与官方"以偏概全"不同，属可优化点。）

2. 答案必含要素（踩分骨架：明确表态 ＋ 论证方法/论据识别 ＋ 分层依据（可取处＋不足处）＋ 效果/结论） - 要素A＝表态并锚定评价维度：合理／不合理、充分／不充分、录用／不录用、能／不能，必须先亮判断。 - 要素B＝识别论证方法与论据性质：举例／对比／引用／类比＋论据是否典型、单一、可比。 - 嘉定-2026-二模第6题（4分）：概括核心观点1分／列举三个论据人物关系1分／阐述三者理论关系1分／论证效果1分。 - 要素C＝充分性/针对性/录用题常有"立场天花板"，且必须结合文本分层，不结合判低分。 - 闵行-2025-一模第22题（第③段反驳是否充分，5分）评分说明："不充分满3、充分最多2。" - 崇明-2026-二模第6题（投稿是否录用）评分说明："不录用最高得3。" - 要素D＝概念一致性视角（论述类特有）：反驳"逻辑严谨"时可攻"概念偷换／内涵不统一"。 - 青浦-2026-一模第7题＝"智力"内涵不统一，专长≠智力是概念偷换。

三、材料关系分析（互补 · 补充 · 递进 · 反驳 · 分歧）

1. 考什么 / 思考动作 面对多则材料，先各自提炼核心观点，再判定它们之间是什么关系（互补／递进／反驳／分歧／印证），并说明"谁补了谁的什么、在哪一点上冲突"。

2. 答案必含要素（踩分骨架：各材料核心观点（分列定额）＋ 关系判定 ＋ 关系点的具体阐释） - 要素A＝各材料观点分列给分：几则材料就几个基础分点。 - 徐汇-2025-一模第7题（4分）："写出一点1分，两点3分，三点4分"——补充点越多越得分，宁多勿漏。 - 要素B＝明确关系词并落到具体交叉点：互补＝各补对方缺口；分歧＝指出分在哪一具体判断上。 - 青浦-2026-二模第6题（4分）＝热量赤字 vs 平稳血糖降胰岛素，分歧在"是否区分热量来源"——分歧题必须把冲突聚焦到一个可指认的争点。

四、概念题（下定义 · 名词解释 · 词语在文中所指 · 用词是否恰当）

1. 考什么 / 思考动作 把文中散布的属性收拢成"种差＋属"的判断句（下定义），或指认某抽象概念在文中具体指代什么，或判断某词用得准不准并给替换。

2. 答案必含要素 - 下定义型（核心属性/种差（可拆多点）＋ 上位属 ＋ 定义句式规范）——见范例卡长宁第3题。 - "在文中所指/含义"型（表层义 ＋ 文中特定所指（分点）） - 嘉定-2026-二模第3题（2分）："'人类的主体性经验'1分，'科学'与'被忽视/遮蔽'1分。答'看不见的部分'等不赋分"——必须回填文本语境，泛泛字面义零分。 - 用词恰当性型（指出误用词的正确义 ＋ 给出修改）——见范例卡闵行第3题。

五、推理补全（补三段论 · 补前提 · 推断隐含内容 · 顺序推断）

1. 考什么 / 思考动作 顺着作者的逻辑链，把被挖空的一环补回去——补大/小前提或结论，补作者未明说但可推断的内容，或判断几则材料/几个问题的先后顺序并给逻辑理由。

2. 答案必含要素（踩分骨架：逐空定额，内容须与上下文逻辑严丝合缝） - 要素A＝补三段论/前提：大前提、小前提各自独立给分，须符合演绎形式。 - 黄浦-2025-二模第4题（2分）：大前提、小前提各1分。 - 要素B＝推断隐含内容：依据文本"暗示处"反推，逐点给分。 - 虹口-2026-二模第6题（4分）：④段暗示对"新特点"把握不具体全面；⑤⑥段暗示缺对数字资本主义逻辑的剖析——两方面对应两个给分块。 - 要素C＝顺序推断：给出"两种排序均可"时，得分全在逻辑理由（时间／认知递进／生理→理论→策略）。 - 杨浦-2025-二模第6题＝两种排序可接受，理由依"生理机制→认知理论→学习策略"逻辑。（证据有限：多为答案要点。）

六、概括提炼（内容提要 · 摘要 · 观点提炼 · 拟标题）

1. 考什么 / 思考动作 把长文压缩：提要/摘要要在限定字数内覆盖"研究对象＋核心观点＋分述要点"；拟标题要抓住全文统摄性话题。

2. 答案必含要素（踩分骨架：覆盖面（各分述要点分列）＋ 主旨句 ＋ 字数/文体规范） - 要素A＝分述要点逐条覆盖，点数≈分值。 - 崇明-2025-一模第7题（拟写推荐提纲，5分）＝研究对象、理论支点、写作路径、阅读意义，"各2分"。 - 要素B＝主旨/结论句不可缺：提要须落到全文最终主张。 - 要素C＝字数与文体限制：超/欠字数、缺"提要/导读"体式会扣分（多题标"≤200字""150字内"）。

七、情境迁移与应用（发言提纲 · 倡议书 · 推荐语 · 情境劝说 · 可行性评析 · 据材料举例佐证）

1. 考什么 / 思考动作 把文本观点搬进真实交际情境：写发言提纲/倡议书/推荐语、劝说某方、评析某设想的可行性、替某人反驳、或为某观点举例佐证。既要用对文本观点，又要合乎身份、文体、对象。

2. 答案必含要素（踩分骨架：观点/内容要素逐条（回扣文本核心概念）＋ 文体格式或身份/对象意识 ＋ 语言得体——"双轨给分"） - 要素A＝内容要素逐条：每个建议/理由/规范独立给分，且须回扣原文核心概念，脱离文本判低分乃至0分。 - （分档量表）浦东-2025-二模第7题（5分）设4—5/3/2/0—1分档，"脱离甲乙两文/答非所问判0-1分"。 - 要素B＝文体格式 / 身份 / 对象意识单独占分（论述类迁移题最易丢的隐性分）。 - 奉贤-2026-二模第7题（倡议书，4分）："符合倡议书文体、有呼告语气"1分；青浦-2026-二模第7题（说服妈妈，5分）："第一人称委婉征询"。 - 要素C＝建议须具体可操作：空泛不给分；建议之间有逻辑关系可加分。 - 要素D＝可行性评析型：先概括"设想本质"，再对齐材料需求，判"一致/矛盾"。

八、论述语言赏析（严谨性 · 限定词 · 理性与感性）

1. 考什么 / 思考动作 分析论述文（而非文学文）的语言：限定词/关联词如何保证表意严密，术语与文学性词汇如何兼顾理性与感染力。落点是"论证的严谨/说服力"，而非文学的形象美。

2. 答案必含要素（踩分骨架：语言特征（限定词/关联词/长句/术语）＋ 对论证严谨性或说服力的作用 ＋（如涉感性）情感/感染效果） - 要素A＝限定词/模糊限制语的严谨功能。 - 宝山-2026-一模第6题（4分）＝表表面化与不确定、揭渊源、使表达严谨。 - 要素B＝长句/关联词与逻辑严密、术语与文学性词汇的分工。 - 虹口-2026-一模第6题（4分）＝长句表意严密／关联词论述严谨／文学性词语增感染力——理性（严谨）＋感性（感染）双落点。 - 要素C＝学术语体规范判断（论述类特有的元评价）：语言是否科学、严谨、温和。 - 长宁-2026-一模第6题（6分）＝学术争鸣应科学严谨温和，改文夸大、攻击性强、不符规范。

附：论述类应试通则（自评分说明反推）

  1. 分层即得分：思路题/概括题/材料关系题，逻辑环节数≈分值，逐点1分（长宁第6题"材料二1／材料三2／互补1"；浦东第7题"现状/原因/方法/总结/格式各1"）。
  2. 评价题＝表态＋分层依据＋结合文本，且立场影响得分上限（闵行-2025-一模22"不充分满3、充分最多2"；崇明-2026-二模第6题"不录用最高3"）；两则材料"哪个更合理"须两方观点各2分再褒此贬彼（闵行第7题）。
  3. 迁移/情境题双轨给分：内容要素（回扣文本核心概念）＋文体格式/身份对象意识（浦东第7题"格式1分"；崇明第7题"空泛建议不给分"）。可行性评析＝设想本质＋材料标尺＋逐条比对（长宁第7题）。
  4. 概念题回填语境：下定义须成"是一种……"判断单句（形式本身占1分，长宁第3题）；用词恰当题须训字义＋指语境冲突＋给替换（闵行第3题）；"所指/含义"须落到文中特定内容，字面泛答不赋分。
  5. 补前提/推断题：把作者用"反例/否定"说的话翻转成"正面成立条件"，逐空定额（闵行第4题两前提各1分）。
  6. 论述类与文学类的分野：论述语言赏析落"严谨/说服力"，论证方法（举例/对比/引用/类比）须单独指认并计分。

文学类（现代文阅读二）

对应上海卷"现代文阅读二"：散文 / 小说。

本章重写目标——完全自包含。 每个题型下的"范例卡"都从真题原卷中抄录了【所依托文本概括】【题目原文】【官方答案】【评分说明】，读者不必回翻试卷即可看懂每一道题考什么、怎么给分、怎么套模板。

本次范例卡的取材范围（仅限已逐字读过的 4 份 2026 二模原卷）： - 虹口·散文《夜食》（周荣池）——深夜为远道朋友张罗路边烧烤夜食，由此勾起回忆：童年父亲夜里就酒吃冷肉、少年"我"趁父亲不在吃凉茶泡饭配盐霜萝卜干、青年在盐城用煤油炉煮杂烩菜，直到如今仍爱做杂烩菜给女儿。以"夜食"串起清贫岁月里的温情与生命况味。 - 闵行·小说《女儿的八千里徒步之旅》（黄宗之）——女儿安琦辞职去徒步 8000 里的太平洋屋脊步道，父母从坚决反对、日夜担忧，到中途探望、心疼，最终理解并学会放手；女儿在荒野中寻找真正自由与人生意义。明显的双线结构。 - 宝山·自传散文《我和我的母亲》（陈行甲，选自《在峡江的转弯处》）——回忆童年随寡母上山打柴、深夜喊工、主动分担背柴，直到某次在山顶俯瞰远处的家而顿悟"这座山是我的"。刻画勤劳谦卑坚韧慈爱的母亲，与"我"的成长担当。 - 浦东·小说《托塔天王》——"我"带偏瘫失语的老父亲去看《哪吒之魔童闹海》。父亲当年是电影院烧锅炉的工人、力能单手举起推煤车、是全家的顶梁柱，被"我"神化为"托塔天王"；如今被脑梗（文中戏称"海夜叉""无量仙翁"）夺去神力。以哪吒 / 托塔天王的神话意象贯穿父子两代的守护与传承。

贯穿全章的采分总规律（下列范例卡的评分说明反复印证）： - 赏析题 ＝ 手法 ＋ 内容分析 ＋ 效果 ＋ 情感 / 主旨，且"只贴标签不分析"要扣分（闵行 8："若只谈语言特点、不作分析，角度分最多 2 分"）。 - 作用 / 构思题 ＝ 内容作用 ＋ 结构作用（线索 / 铺垫 / 呼应 / 承上启下）＋ 主题作用，多为"1 点 1 分、点数 ≈ 分值"（闵行 10、闵行 11、浦东 9、浦东 11）。 - 视角 / 差异题要分维度铺开、每维定额（闵行 9："来源差异、展现的目的差异、照片与亲眼所见差异、视角与关注点差异各 2 分"）。 - 情感 / 主旨落点几乎是所有赏析、构思题的固定收束——漏写主题几乎必扣分。

一、语言 / 句子赏析（修辞、词语、句式）

1. 考什么 / 思考动作

指定一处画线句或加点词，问"表达效果 / 语言特点 / 表现力 / 请赏析"。思考动作：先指认手法（比喻、比拟、反语、反复、通感、动词 / 形容词炼字、长短句 / 整散句）→ 落到文本内容（写出了什么对象的什么特点）→ 说效果（生动 / 画面感 / 节奏 / 化抽象为具象）→ 归到情感或主旨。

2. 答案必含要素（踩分骨架 ＝ 手法 ＋ 内容 ＋ 效果 ＋ 情感）

  • 要素 1 ＝ 手法判断（必写，精确命名，如"就近取譬""拟人""反复""化抽象为具象""长短句结合"）＝ 1 分。
  • 要素 2 ＝ 结合文本的内容分析（该手法写出对象的什么特点）＝ 1 分。
  • 要素 3 ＝ 表达效果（生动形象 / 画面感 / 节奏感 / 化抽象为具象）＝ 1 分。
  • 要素 4 ＝ 情感 / 主旨落点＝ 1 分。
  • 扣分红线：光贴标签、不作分析，给不满（见下方闵行 8 评分说明）。

二、描写作用（景物 / 环境 / 细节 / 场景）

1. 考什么 / 思考动作

问某段景物、环境、细节或场景描写"在文中的作用 / 妙处"，或"某处细节似与主事无关，为何要写"。思考动作：内容层（写了什么、营造什么氛围）→ 结构层（铺垫 / 引出下文 / 伏笔 / 承上启下 / 呼应）→ 人物层（衬托形象心理、体现成长）→ 主题层（点题 / 深化主旨 / 奠定基调）。

2. 答案必含要素（＝ 内容 / 氛围 ＋ 结构 ＋ 人物 / 主题，多维铺开）

  • 要素 1 ＝ 内容概括 ＋ 氛围 / 基调＝ 1 分。
  • 要素 2 ＝ 结构作用：引出下文 / 铺垫 / 伏笔 / 承上启下 / 呼应＝ 1 分。
  • 要素 3 ＝ 人物作用：烘托 / 衬托形象或心理、体现成长＝ 1 分。
  • 要素 4 ＝ 主题作用：点题、深化主旨、奠定感情基调＝ 1 分。
  • 该题型典型特征是"广撒网、多维给分、每维 1 分"，凑够分值即止，宁可多写几维择优。

三、段落作用与构思（线索 / 反复意象 / 独句成段 / 双线结构 / 围绕标题选材）

本章证据最富集、评分说明最清晰的一节。

1. 考什么 / 思考动作

问"某段在构思上的作用""多次出现的 X 的用意""独句成段的妙处""双线结构的构思""围绕某意象 / 标题选材组材的构思"。思考动作：先判定构思手法（线索、双线、伏笔、对比、回环、由实入虚、成年视角回望、独句成段）→ 内容作用 → 结构作用（贯穿 / 呼应 / 铺垫 / 承上启下 / 蓄势推动高潮）→ 情感递进 → 主题深化。

2. 答案必含要素（核心 ＝ 构思手法 ＋ 结构作用 ＋ 情感 ＋ 主题）

  • 要素 1 ＝ 构思手法判断：线索、双线、对比、回环、独句成段、由实入虚、成年视角……＝ 1 分（构思题此点几乎必踩）。
  • 要素 2 ＝ 结构作用：贯穿全文使结构严谨 / 串联材料 / 前后呼应 / 铺垫伏笔 / 推动情节高潮＝ 1 分。
  • 要素 3 ＝ 内容 / 情感作用：情感递进、心理转折＝ 1 分。
  • 要素 4 ＝ 主题作用（升华 / 深化主旨）＝ 1 分，必写。
  • 双线题专用：必须"分别写两条线的内容 ＋ 写各自推进变化 ＋ 写两线交织及主题"（见闵行 11 评分说明，三块各 2 分）。

四、标题意蕴与关键词 / 句含义（含义理解）

1. 考什么 / 思考动作

问"标题 X 的意蕴""某词（如'谎言''胎记'）在文中的含义""某句的内涵"。思考动作：表层义（字面 / 实指）→ 深层义（象征 / 比喻 / 双关）→ 情感 / 主旨义，层层剥。

2. 答案必含要素（＝ 表层义 ＋ 深层 / 象征义 ＋ 情感主旨义，分层给分）

  • 要素 1 ＝ 表层 / 实指义＝ 1 分。
  • 要素 2 ＝ 深层 / 象征 / 双关义（往往不止一层，逐层各给分）。
  • 要素 3 ＝ 情感态度 / 主旨指向。
  • 与第三节"标题构思题"不同：意蕴题重"含义分层"，构思题重"线索 / 选材 / 呼应"。含义题拒绝只答字面，只写字面义通常不赋分。

五、叙述视角与人称

1. 考什么 / 思考动作

问"以特殊身份的'我'作叙述者的效果""不同叙述视角的赏析""同一对象在不同视角下呈现不同面貌的原因""叙述视角转换的作用"。思考动作：先定视角类型（第一 / 第三人称、限知 / 全知、成年回望、当事人 vs 旁观者）→ 真实感 / 代入感 / 客观距离感 → 内心刻画 → 情节（悬念 / 串联）→ 主题深化。

2. 答案必含要素（＝ 视角 / 人称类型 ＋ 叙事效果 ＋ 主题作用）

  • 要素 1 ＝ 视角 / 人称身份判断（叙述者个性、第几人称、限知 / 全知、成年 / 童年）。
  • 要素 2 ＝ 叙事效果：真实感 / 代入感 / 客观距离感 / 悬念 / 内心刻画。
  • 要素 3 ＝ 视角转换或视角差异的整体效果（若考转换 / 对比）。
  • 要素 4 ＝ 主题深化。

六、形象概括

1. 考什么 / 思考动作

问"概括 X 的形象 / 特征""结合某人的话梳理 X 的美丽""评价 X 这一人物"。思考动作：分维度（外貌 / 身份 ＋ 性格 ＋ 品德 / 精神 ＋ 才智），逐维配文本依据；若题目要求"如何塑造"则先答塑造手法（正侧结合、对比、细节）。

2. 答案必含要素（＝【塑造手法 ＋】多维特征词，逐维分列定额）

  • 要素 1（视题而定）＝ 塑造方法：正面描写 ＋ 侧面烘托、对比、细节（问"如何塑造"时须答）。
  • 要素 2 ＝ 多维特征词：外貌 / 身份、性格、道德、才智 / 精神……逐维一词一分，宁多勿漏、不可堆同义词。
  • 若题干已给出分类框架（如"长相 / 性格 / 道德 / 智慧"），必须逐框对应作答，一框一分。

七、评价类（是否可删 / 是否矛盾 / 是否重复 / 是否突兀 / 是否削弱主题 / 道理是否含蓄隽永）

1. 考什么 / 思考动作

问"某段可否删去""前后两处是否矛盾""结尾与前文是否重复""结尾是否突兀""某描写是否削弱主题""针对某种看法评价人物""这篇小说道理含蓄隽永还是粗浅乏味"。思考动作：先明确表态（不可删 / 不矛盾 / 不重复 / 看法不成立 / 含蓄隽永……）→ 从内容、结构、主题分层给出理由 → 收束到效果 / 意义。

⚠️ 关键分型：先看这道题在评价"什么"。 - 评价"某段 / 某处内容可否删、是否矛盾" → 落到内容 / 结构 / 主题三维（"可删否"是作用题的变体）。 - 评价"这篇小说讲道理的方式是否高明（含蓄隽永 vs 粗浅乏味 / 说教）" → 不要套内容主题维的"对比 / 象征 / 标题双关"，而要落到"叙事艺术如何让道理自然生成"：从环境 / 人文背景写起 → 节奏舒徐、不急于说理 → 结尾说理切合人物身份与语境（借情节自然带出，非作者跳出来议论）→ 如盐化水、画龙点睛。判错这个分型，会答成主题解读、全盘偏离采分点（见下方静安《壶友》范例卡的踩坑复盘）。

2. 答案必含要素（＝ 表态 ＋ 分层依据 ＋ 效果；注意"立场决定得分上限"）

  • 要素 1 ＝ 明确表态 / 立场（多数需第一句亮明）。
  • 要素 2 ＝ 结合文本的分层依据：
  • 若评价"内容可删 / 矛盾"→ 内容作用 ＋ 结构作用 ＋ 主题作用（"可删否"本质是作用题的变体）。
  • 若评价"讲道理方式是否含蓄"→ 环境 / 背景铺垫 ＋ 舒徐节奏不急说理 ＋ 说理切合人物身份语境 ＋ 如盐化水的效果（叙事艺术维，非主题维）。
  • 要素 3 ＝ 效果 / 意义收束。
  • 关键规律——立场影响得分上限：命题往往预设某一立场更优，评分设不对称封顶（"支持前者详细赋分、支持后者酌情赋分"），应试时优先选高分立场。

八、文学短评与综合评析（结合评论观点 / 理论卡片 / 情理关系作答）

1. 考什么 / 思考动作

给一段评论观点（"用最轻松的口吻写最沉重的内容"）、一段理论卡片，或要求就"人 / 事与情 / 理的关系"写综合评析。思考动作：先回扣评论 / 理论的核心说法 → 落到本文指定段落的具体语句 / 内容 → 展开分析评论（须命中给定说法规定的落点）。

2. 答案必含要素（＝ 观点 / 术语回扣 ＋ 文本具体内容 ＋ 依给定角度的分析评论）

  • 要素 1 ＝ 具体内容（引 / 概述指定段落的原句）。
  • 要素 2 ＝ 分析评论（占大头，且必须紧扣题目给定的评论观点 / 理论概念）。
  • 踩分红线：脱离所给评论或所给文本、只泛泛赏析，判低档。

古典诗歌（诗歌鉴赏）

本章的读法：每个二级题型都按"考什么 → 答案必含要素 → 自包含范例卡"三段展开。范例卡自带诗歌原文、题目原文、官方答案、官方评分说明，读者无需翻原卷即可看懂"这道题为什么这样给分"。范例全部取自本次通读的 5 份原卷（长宁 2026 二模、崇明 2026 二模、闵行 2026 二模、青浦 2026 二模、虹口 2025 一模），优先选用评分说明清晰的真题；评分说明缺失处已注明"证据有限"。

贯穿全类型的采分骨架：诗歌赏析题几乎都可拆成三层——手法/切入角度 → 逐联(逐层)内容分析 → 情感/主旨落点。分点定额，点数≈分值；"情感/主旨落点"几乎是所有题的固定收束，漏则失分。评价类另有硬规则：立场影响得分上限（见题型六）。

本次 5 份原卷的诗歌真题分布：炼字（闵行·醉）、结构（虹口·以景语起收）、手法（崇明·尾联）各 1 道，评价择优（长宁·两版本）1 道，风格印证（青浦·悲中有壮）1 道；无两诗比较题。

题型一 · 炼字炼词（含"字眼统摄全诗"）

1. 考什么／思考动作 指定一字/一词（多为动词、色彩词，或统摄全诗的"诗眼"），赏析其表达效果。思考动作：还原该字描写的对象与画面 → 指出手法（动静/明暗/化虚为实/夸张/用典等）→ 落到意境与情感。若是"诗眼/主题字"（如"醉""愁"），须以该字为纲串起全诗内容，逐层说清"哪一层醉在哪几句、用了什么手法"。

2. 答案必含要素 - 诗眼统摄型 ＝ 醉（字）的内容 ＋ 表现该字的手法（必须结合内容），两栏定额、各占约半。见下卡评分说明。 - 落笔要同时回答"写了什么内容"和"用什么手法写"，只答其一失一半分。

题型二 · 情感层次 / 情感（风格）赏析（主力题型）

1. 考什么／思考动作 题干常见两类：一是"情感丰富而有层次/情感如何层层推进"；二是给一句风格评语（如辛弃疾"悲中有壮"），要你结合全诗印证。思考动作：逐联（首→颔→颈→尾）扫描，每联指认其抒发的具体情感及所借手段（直抒/借景/用典/虚实）；若是风格评语，则把评语拆成两组情绪（如"悲"与"壮"），分别指认它们各落在哪些联，最后用一句概括其"交织/推进"关系。

2. 答案必含要素（最稳定的采分模型：逐联/逐组分点 ＋ 情感落点总结） - 每联须写出"情感 ＋ 抒情手段"；风格印证型须把评语两端（悲/壮、乐/哀）各自坐实到具体联，再点出二者关系。 - 关键情感点常设为"必答"：漏掉评语的核心一端（如只写"悲"不写"壮"），得分会被压。 - 本型在这 5 份原卷中仅青浦一道，且未附评分说明（证据有限）；下卡的分栏为据答案结构反推。

题型三 · 手法赏析（以景结情 / 情景交融 / 虚实 / 用典 / 设问 / 乐景衬哀）

1. 考什么／思考动作 指定某联（多为尾联）或全诗某手法，赏析。思考动作：准确判断手法（可多个并列）→ 落到具体诗句做内容分析 → 说效果（意境/画面/时空拓展）→ 收到情感或主旨。"以景结情"须额外点出"以景收束、留想象空间、含蓄不尽"。

2. 答案必含要素（采分骨架 ＝ 手法判断 ＋ 逐点内容分析 ＋ 效果 ＋ 情感落点，逐点定额） - 多种手法并列时，每点定额；核心手法点数高（1 点 2 分），辅助点数低（1 点 1 分）。 - 每报一个手法都要贴着诗句解释它怎么用，并说清它带来的效果，最后落到情感。

题型四 · 结构赏析（起承转合 / 首尾 / 以景语起收 / 结尾一联）

1. 考什么／思考动作 针对全诗章法（起承转合）或首尾照应（"以景语起、以景语收"）赏析。思考动作：分别赏析起句与结句（或某联在"承/转"中的位置）→ 点明二者构成的情感转折（如由乐入哀）→ 说结构与抒情效果。

2. 答案必含要素（首赏 ＋ 尾赏 ＋ 点明情感转向 ＋ 效果，四栏定额） - 起、结两处分开赏析且分值不均（起句权重更高）；中间必须点出情感由乐转哀（或转入某情）的枢纽，这一分最易漏。 - 结尾落到"效果"（情感淋漓、深挚感人之类）。

题型五 · 两诗（词）比较

1. 考什么／思考动作 两首诗对读，比较情感、意象、手法、意境或虚实处理的异同。思考动作：先找相同点（母题/情感/手法层面）→ 再逐诗指出相异点 → 落到各自的情感或写作意图。

2. 答案必含要素（相同点定额 ＋ 各诗相异点分列定额） - 稳定采分模型：共同点 1–2 分 ＋ 甲诗特点 2 分 ＋ 乙诗特点 2 分；比较必须"两诗都说到"，只析一首必失半分。 - 若含"孰优"判断（比较＋评价），需再加"结尾/结构效果比较"一栏，认同或不认同均可给分。

题型六 · 评价类（评语是否成立 / 两版本择优 / 归类 / 印证型）

1. 考什么／思考动作 给出一句评语、两个字词/句子版本、或两种归类，判断并说理。思考动作：先明确表态/立场 → 结合具体诗句分层论证 → 收到评语所指的效果或意义。择优题须同时分析所选版本之优与另一版本之劣。

2. 答案必含要素 - 择优/两版本型 ＝ 所选版本句意情感分析 ＋ 在全诗中的效果 ＋ 与另一版本的优劣比较，三栏各占约 2 分（见下卡）。 - 两个版本均可选：官方给出对立的两份满分示例，关键是把所选版本讲透并踩落另一版本，而非猜"标准答案"。 - 立场影响得分上限（硬规则，旁证）：部分评语题"不认同"更好写、封顶更高（如浦东 2025 一模：同意最多 3 分、不同意满 5 分；金山 2025 一模：赞同满 5、不赞同最多 4）。作答前先掂量哪个立场"料多、上限高"。

文言记叙（文言文一）

本章聚焦上海卷"文言文一"（以人物传记、史传节选为主的记叙类文本）的主观简答题。 每个二级题型固定写成三块：①考什么/思考动作；②答案必含要素（从官方评分说明反推）；③自包含范例卡——直接抄录真题的文本背景、题目原文、官方答案、评分说明，并用箭头拆解每个要素对应哪个采分点。 范例卡全部取自四份原卷：青浦-2026-二模《方琼州传》、宝山-2026-二模《画网巾先生传》、长宁-2026-二模《志衍传》、虹口-2025-一模《范滂传》。断句、翻译、词义三类技能题同样配范例卡（附文言原句+采分点）。样本单薄或所读四卷无对应真题处，均标注"证据有限"。

一、人物形象概括

1. 考什么 / 思考动作

给出人物在传记中的言行事迹，要求"概括其形象 / 特征 / 品格 / '奇'在何处"。思考动作：先分类扫描事迹（为官—治理、为人—品德、为学—才识、待人—交友、临难—气节），再为每一类提炼一个特征词，事迹与特征词一一挂钩。切忌只堆特征词而不落事例，也切忌只复述事例而不提炼词。

2. 答案必含要素（核心）

  • 要素A｜点数≈分值，逐点罗列。 4 分题列四点、5 分题列四点封顶，一点一个"表现+概括"。作答宁多勿漏。
  • 要素B｜"表现（事迹）+ 特征"成对出现。 只叙事不概括、或只给词不给事，均扣分。
  • 要素C｜多维度扫描，避免同质重复。 一个人物要从不同侧面（气节、才识、治绩、待人）取材，不要四点都在说同一件事。
  • 要素D｜心理 / 神态类形象题给"神态词 + 层次递进"。 见范例卡二。

二、选材组材 / 整体性分析

1. 考什么 / 思考动作

问"本文选取了哪些材料 / 为何这样组材 / 前后联系如何形成整体感"。思考动作：先列作者选了哪几组事迹，再问"为什么选这些、为什么这样排"——指向同一形象内核的多角度支撑、正面事迹与侧面烘托的搭配、乃至如实记录缺点以求'实录'。

2. 答案必含要素（核心）

  • 要素A｜"多角度选材"要点清。 从才学 / 仪态 / 政见 / 军事等多侧面取材 + 侧面烘托 + 如实记录缺点（体现史传"实录"）。
  • 要素B｜"组材逻辑"要点明映衬 / 铺垫作用。 说清前一组材料如何映衬、铺垫后一组。
  • 要素C｜"整体性 / 前后联系"题按"指认现象 1 分 + 结合文本分析加分"给分，空泛不给分。

三、塑造手法（正侧结合 / 补叙 / 细节 / 侧面烘托）

1. 考什么 / 思考动作

问"某段 / 全文是如何塑造人物的""某处补叙有何作用""某细节 / 侧面描写的作用"。思考动作：先判定手法（正面：语言 / 动作 / 神态 / 外貌；侧面：他人反应、环境衬托、对比烘托；补叙 / 补记），再分析该手法揭示了人物哪几个侧面，最后落到形象特征。

2. 答案必含要素（核心）

  • 要素A｜塑造手法题＝方法（1 分）+ 分析（2 分）+ 形象概括（1 分）。 这是最硬的官方骨架，三块缺一不可。
  • 要素B｜"方法"要正侧兼顾。 既指出正面描写（写了哪些直接事迹），也指出侧面烘托（他人反应）。
  • 要素C｜"分析"要落到具体事例，"形象概括"要给出品格词。

四、评价人物 / 治理能力 / 行为动机

1. 考什么 / 思考动作

问"评价其治理能力""是否堪比古之良吏""为何做出某种选择 / 有何动机""是否认同对某人的评价"。思考动作：先明确表态或定位，再从不同侧面（治理：慎刑断案、爱民兴学、因地制宜、不徇私；动机：从人物自述与处境中分层提炼）分点举证。

2. 答案必含要素（核心）

  • 要素A｜治理能力题＝治绩分维度分点，任一点定额封顶。 见范例卡一（任一点 2 分，满 5 分）。
  • 要素B｜每一维＝"哪件事 + 体现什么能力/品格"，扫描全篇不同官职阶段。
  • 要素C｜动机 / 原因分析题＝从人物自述逐层拆解，一层一点。 见范例卡二。

五、评析史传客观性 / 作者身份影响 / 私传阅读态度

1. 考什么 / 思考动作

近两年（尤其 2026）显著新增的"元评价"题型：问"作者身份对文章客观性的影响""对古代私传应持怎样态度""某句史评是否得当"。思考动作：先明确史传/实录应有的"客观—褒贬并存—征实"标准，再指出该文因作者身份/私传性质可能造成的偏向（有褒无贬、失真风险），最后给辩证态度（既承认独特价值又警惕局限）。

2. 答案必含要素（核心）

  • 要素A｜态度必须辩证：不盲从尽信、也不全盘否定（1 分）。 单边表态直接压分。
  • 要素B｜"失真风险"一侧：指出私传/身份带来的情感倾向与失实可能，并举文中/补充材料证据。
  • 要素C｜"独特价值"一侧：指出私传保留鲜活细节、呈现人物复杂性、可补正史之阙的价值，并举文中证据。
  • 两侧证据均衡给分（各占定额）。

六、句读断句（技能型·采分模板）

1. 考什么 / 思考动作

在无标点的文言语段中断句，通常 3—5 处。思考动作：抓虚词（者/也/矣/乎/焉/而/则）、抓对话标志（曰/云）、抓句式对称与主谓完整、抓人名官职专名不可断开。

2. 答案必含要素（核心·采分模板）

  • 计分方式＝"错一处扣一分，扣到 0 分为止"。 多断、漏断、错断均按处扣分——只在有把握处下笔，切忌为凑数硬断。
  • 作答口诀：①先通读定大意、锁定人物与动作主语；②每个完整"主+谓（+宾）"后优先考虑断；③句末虚词后断、句首虚词（夫/盖/故/是以/岂）前断；④"曰/对曰"后断；⑤人名、官名、地名连读不断。

七、翻译（技能型·采分模板）

1. 考什么 / 思考动作

翻译传记文中画线句（多 4—7 分）。思考动作：逐字对译、圈定关键实词与特殊句式（判断/被动/宾语前置/省略），补出省略主语，落实每个采分点。

2. 答案必含要素（核心·采分模板）

  • 计分方式＝"关键实词 / 每分句为采分点，一点 1 分（难句可 2 分）"。 漏译或错译一个采分词即扣一分。
  • 作答步骤：①切分为若干分句/短语；②每个分句锁定 1 个关键实词或句式作为踩分锚点；③补省略主语（用括号）；④直译为主，句式标志（被动"为…所"、判断"…者…也"、宾语前置）须译出；⑤检查每个采分点是否落地，人名官名照抄不译。

八、词义解释（技能型·采分模板）

1. 考什么 / 思考动作

解释加点实词、或为加点词选择正确义项（通常各 1 分）。思考动作：据传记语境定一词多义、注意词类活用与古今异义，用现代双音词精准替换。

2. 答案必含要素（核心·采分模板）

  • 计分方式＝逐词给分、格式化，每词 1 分。 评分只认精准命中的核心义，泛化表述零分。
  • 作答模板：①代入语境判断词性（动/名/形），先定活用；②在常见义项中选贴合上下文动作逻辑的一个；③用一个精准现代词作答，不写句子、不作解释性长句；④警惕古今异义与偏义复词。

附：本章通用采分规律（据官方评分说明归纳）

  1. 点数≈分值：概括类、治绩类、动机类均"逐点给分（1 分或每点 2 分封顶）"，作答分点罗列、宁多勿漏。
  2. 形象/塑造题三件套：方法（1）+ 分析（2）+ 形象概括（1）——虹口2025一模19为标准模。
  3. 评价/治理题：分维度举证、任一点定额封顶，每点"四字概括 + 具体事迹"成对——青浦2026二模19。
  4. 动机/原因题：从人物自述与补充材料逐层拆解，一层一点、每点绑文本——宝山2026二模18。
  5. 史传客观性题（2026 新增高频）：辩证表态（1）→ 失真风险举证（2）→ 独特价值举证（2），两侧均衡——长宁2026二模19。
  6. 技能三题：断句"错一处扣一分，扣到 0 分为止"；翻译"关键实词/分句一点 1 分、难句 2 分、特殊句式须译出结构"；词义"逐词精准命中核心义，泛化表述零分"。

证据充足度：人物形象概括、心理神态、塑造手法、评价治理、动机分析、史传客观性、断句、翻译、词义九类均有本章四卷内官方评分说明支撑，范例卡自包含可靠。唯"选材组材/整体性分析"在所读四卷文言文一中无对应真题，标注证据有限，仅保留要点级骨架。

文言论说（文言文二）

本章定位：文言文二多为"论/记/序/书"等论说类文本，主观题集中考"论证与说理分析、句式赏析、双文本比较、评价观点与思想意义、辩驳技巧、记序书缘由与写作意图、为观点提供证据"七种题型。 自包含说明：为解决"只报'某区第几题'却不给文言原文和题干、读者看不懂"的问题，本章每个题型下设「自包含范例卡」，直接抄录文本背景 + 题目原文 + 官方答案 + 评分说明 + 模板拆解。所有范例卡均取自实际读到的四份 2026 二模原卷：闵行（汀州卧龙书院记·记）、青浦（甲乙下第赠序·序）、长宁（周象九五十寿序·序）、浦东（壑舟记·记）。 证据边界：这四份原卷的文言文二未出现"论证方法/思路""辩驳技巧""为观点提供证据"三类独立主观题，故题型一、五、七保留"考什么 + 答案必含要素"，范例改用交叉引用或标注「证据有限」，不臆造未读原文。

题型一：论证方法 / 说理思路分析

1. 考什么 / 思考动作

问"某段如何展开论证""概括论述/行文思路""某事例（引用/转述）在论证上的作用"。思考动作：先定位核心观点/论题，再拆论据（举例、引用、类比），再指认论证方法（对比、类比、举例、引用、先立后破），最后落到论证效果（鲜明有力/严密/说服力）。思路题须分层标序（首先→接着→最后），点数≈分值。

2. 答案必含要素（从评分说明反推）

  • 单段"如何论证"= 举例/论据分析 + 点明方法 + 效果（三点定额）。
  • "论述/行文思路"= 分层要点，逐点 1 分，点数 = 分值。

题型二：句式 / 说理语言赏析

1. 考什么 / 思考动作

问"从句式角度赏析画线句""赏析某句富有表现力之处"。思考动作：先辨句式特点（整句/散句、整散结合、长短句、短句、骈散、排比、顶真），再作具体分析（哪几处如何整、如何散、哪几句长/短），最后落表达效果（节奏气势/严密灵动 + 表情达意/说理内容）。若题干说"富有表现力"，通常要句式 + 修辞双轨赏析。

2. 答案必含要素（从评分说明反推）

  • 句式赏析三点定额：句式特点 + 整散/长短分析 + 表达效果。
  • "富有表现力"题 = 句式角度 + 修辞角度双轨，各半分值。

题型三：两文比较（说理特色·章法·写作意图·文体）

1. 考什么 / 思考动作

甲乙（或甲乙丙）双文本，问"说理特色异同""章法差异""写作意图异同""写法不同的原因""两种文体特点"。思考动作：先抓共同点，再逐文分列相异点；"原因"类须归因到文体、写作目的、交际情境、作者身份。

2. 答案必含要素（从评分说明反推）

  • 比较题固定骨架 = 相同点 + 甲/乙各自相异点分列（相同 1 层、甲乙各 1 层，分列定额）。
  • "章法/文体差异"须点明"文体决定章法/写法"。

题型四：评价观点 / 思想意义（评析类）

1. 考什么 / 思考动作

问"提炼并评析××观点/使命观的思想意义""这段议论说理，加以评析""是否认同文末观点"。思考动作：明确表态/立场 → 概括观点内涵 → 结合文本事例分层依据 → 落到历史意义 + 现实意义（或表达效果）。注意立场影响得分上限。

2. 答案必含要素（从评分说明反推）

  • 思想意义评析三段式：观点解释 + 历史意义 + 现实意义。
  • "评析一段议论说理"= 评价定性 + 内容梳理 + 评价理由。
  • "是否认同"类 = 观点理解 + 事例分析 + 评价。

题型五：辩驳技巧分析（以退为进 / 先破后立 / 反问）

1. 考什么 / 思考动作

问"分析××'以退为进'的辩驳技巧""写'或者'质疑的作用""如何批评/反驳对方"。思考动作：还原辩驳的推进步骤（先退让/自抑 → 再转/立论），指认技巧名称（以退为进、先破后立、树靶反驳、反问讽刺），点明说理效果。

2. 答案必含要素（从评分说明反推）

  • 拆解辩驳的动作链，还原"退—进"或"破—立"层次。
  • "反驳是否充分/有力"类（元评价）须分档，且立场影响得分上限。

题型六："记 / 序 / 书"体缘由与写作意图分析

1. 考什么 / 思考动作

"记"体文问"书院记为何提某事""为何反复写某风俗""叙述得名来由的作用"；"序/书"问"编撰意图""写信目的""某写法的作用"。思考动作：把看似"离题"的材料与记述对象（建筑/人物/事件）建立关联，指向作者情志、主旨、铺垫。

2. 答案必含要素（从评分说明反推）

  • "记体为何提某事"= 该事与记述对象/主旨的多重关联（关联点开放，任一成立即赋分）。
  • "得名来由/命名缘由"叙述 = 内容概述 + 表情志 + 铺垫。

题型七：为观点提供证据 / 事例能否印证（迁移印证）

1. 考什么 / 思考动作

给一则课外/补充材料或本文另一处论断，问"为某观点提供理由与证据""某事例能否验证/印证某观点""选出不能与上文互证的一项"。思考动作：先准确复述被验证的观点内涵，再将材料/事例逐条对接到观点的各个层面，最后判断契合度。

2. 答案必含要素（从评分说明反推）

  • 为观点提供证据 = 指定例证 + 分层对接观点各层面。
  • "事例能否验证观点"= 材料分析（主体）+ 观点辨析（判断），分值前重后轻。

附：本板块采分通则（据四份原卷评分说明归纳）

  1. 论证/思路题（题型一）：核心观点/论题 + 论据分析 + 方法（对比/类比/举例/先破后立）+ 效果；思路题逐层标序，1 点 1 分、点数≈分值。
  2. 句式赏析（题型二）：句式特点 + 整散/长短/骈散的具体分析 + 表达效果，三点定额；"富有表现力"题按句式角度 + 修辞角度双轨各半分值给分（浦东 21：句式 2 + 修辞 2）。
  3. 比较题（题型三）：相同点 + 甲乙各自相异点分列；相同写文体共性，相异各扣本文最具辨识度的意象/用典（青浦 22：相同 1 + 甲 2 + 乙 2）。
  4. 评价/评析题（题型四）：表态/立场 + 观点内涵 + 结合文本事例分层 + 历史意义/现实意义；思想意义题走"解释 + 历史 + 现实"三段式（闵行 23：各 2 分封顶 5），评析议论段走"评价 + 内容梳理 + 理由"（长宁 22：1+2+2）；立场是否明确、有无深度决定得分上限（浦东 22）。
  5. 辩驳技巧（题型五）：还原"退—进 / 破—立"动作链 + 指认技巧 + 说理效果；"是否充分"类分档给分、立场决定上限。
  6. 记体缘由/写作意图（题型六）：把"离题材料"与记述对象/风格/主旨建立多重关联，关联点开放、任一成立即赋分（闵行 22："关联均可"）。
  7. 迁移印证（题型七）：指定例证 + 分层对接观点各层面；"能否验证"重在材料分析而非判断本身。
