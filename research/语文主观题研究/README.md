# 上海高考语文·阅读主观题研究

从教研角度，对已积累的上海各区高三一模/二模阅读主观题做分类整理，产出**分文体的答题模板库**（每类题"答案应包含哪些要素"）。与选择题教练 `chinese-mcq-coach` 配套。

## 研究章程

- **一级分类 = 5 大文体**（跟上海卷五个阅读板块走）：
  1. 论述类（现代文阅读一：议论/科技说明/非连续文本）
  2. 文学类（现代文阅读二：散文/小说）
  3. 古典诗歌（诗歌鉴赏）
  4. 文言·记叙（文言文一：人物传记为主）
  5. 文言·论说（文言文二：论/记/序/书）
- **二级 = 该文体下的高频题型**（各文体题型清单不同）。
- **三级 = 答题模板**：思考动作 + 答案必含要素清单 + 真题例证。
- **方法**：实证驱动。模板骨架从**官方评分说明**反推，跨区题目交叉验证，样本不足处标注"存疑"，不硬编。
