# CLAUDE.md — 语文主观题研究（研究方法与关键决策）

本目录是"从考卷语料 → 答题模板库 → Claude Skill"的一次完整教研研究。此文件记录**研究方法、关键决策、以及后续扩充时必须遵守的原则**，供未来在此基础上加语料/加文体/重跑质检时复用。

## 一、研究目标

不是笼统总结"答题套路"，而是产出**分文体、可实证、自包含、经验证**的答题模板库：每种题型明确说清"答案必须包含哪些要素才能拿分"，且每条要素都能回溯到官方评分说明。最终固化为 skill `chinese-subjective-coach`（见 ../../skills/）。

## 二、四条不可动摇的方法论（后续扩充务必遵守）

1. **文体优先，两级分类**：一级=文体（论述/文学/诗歌/文言记叙/文言论说，对应上海卷五个阅读板块）；二级=该文体下的题型。**先判文体再定题型**——同一个"赏析画线句"在不同文体里落点完全不同，混用即错。

2. **答案必含要素从"官方评分说明"反推，不许拍脑袋**：只有 `-原卷.md`（带官方评分说明，逐字）才能作为采分骨架的证据源。`-阅读.md`（答案要点）只能做旁证。凡无评分说明支撑的，必须标注"证据有限"，绝不硬编。

3. **范例卡必须自包含**：读者不跳出本文档就能知道"这道题在问什么"。每张范例卡固定六段——【出处】【所依托文本/材料】【题目原文】【官方答案】【评分说明】【模板拆解】。**踩过的坑**：早期版本只写"某区第X题"、不抄题目原文，导致读者必须翻原卷，被判定为"可读性为零、必须重写"。教训：宁可卡片长，不可让读者跳出文档。

4. **必须做双盲质检验证有效性**：不是核对"抄得对不对"，而是让 agent 拿模板**真做题**，用标准答案验证模板管不管用。防作弊设计——做题 agent 只拿【文本+题干+模板】盲做（看不到答案），阅卷 agent 独立拿【标准答案+评分说明】打分，两者物理隔离。本轮抽 9 题（约占范例卡 30%），得 42/42、100%。**skill 上线前还须做"隔离测试"**：派一个对研究过程零知情的全新 agent，只靠读 skill 文件解一道没进过训练的新题，验证 skill 真正可迁移（本轮通过，满分）。

## 三、执行经验（多 agent 编排）

- **提取/重写/质检都用子 agent 并行**（按文体 5 路 或 按批次分工），主线程只做综合与出盲卷。这是本研究能快速完成的关键。
- **子 agent 会因主线程被用户打断而中途夭折**：长任务优先后台 agent，避免在其运行时打断。
- **子 agent 偶尔以版权为由拒绝逐字复制整卷**：个人教研用途下，抓取原卷用云浏览器 innerText（见 fetch-exam-papers skill），研究阶段只处理"题干+答案+评分说明"不涉整篇复制，可规避。

## 四、文件说明

- `高考语文阅读主观题答题模板库.md` — 主报告（总纲：文体×题型矩阵、跨文体八条采分铁律、双盲质检背书）。
- `章节-*.md` — 五个文体的完整模板（题型→答案必含要素→自包含范例卡）。这五份即 skill 的 references 来源。
- `原始数据-*.md` — 约 300 题的结构化提取（题干/分值/归类/答案要素），按四个批次分档。

## 五、已知缺口 / 后续可做

- **2024届语料（32份）未纳入模板研究**：因该批全是 `-阅读.md`、无一份官方评分说明，无法贡献采分骨架，只影响"题型演变趋势"与"题型清单完整性"两个覆盖面维度，**不影响模板正确性**。若要做"题型逐年演变"研究，需补这批的最早时间锚点。
- 部分区（徐汇/松江等）2025二模题源受限、只有答案要点，相关题型已标"证据有限"。
- 诗歌"两诗比较"、文学"标题意蕴/可删类"在本轮所读原卷中样本不足，范例卡待补——补充时须坚持"有官方评分说明才做实卡"的原则。
