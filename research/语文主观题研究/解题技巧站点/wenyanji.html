文言·记叙（文言文一·人物传记）

本章聚焦上海卷"文言文一"（以人物传记、史传节选为主的记叙类文本）的主观简答题。 每个二级题型固定写成三块：①考什么/思考动作；②答案必含要素（从官方评分说明反推）；③自包含范例卡——直接抄录真题的文本背景、题目原文、官方答案、评分说明，并用箭头拆解每个要素对应哪个采分点。 范例卡全部取自四份原卷：青浦-2026-二模《方琼州传》、宝山-2026-二模《画网巾先生传》、长宁-2026-二模《志衍传》、虹口-2025-一模《范滂传》。断句、翻译、词义三类技能题同样配范例卡（附文言原句+采分点）。样本单薄或所读四卷无对应真题处，均标注"证据有限"。

一、人物形象概括

1. 考什么 / 思考动作

给出人物在传记中的言行事迹，要求"概括其形象 / 特征 / 品格 / '奇'在何处"。思考动作：先分类扫描事迹（为官—治理、为人—品德、为学—才识、待人—交友、临难—气节），再为每一类提炼一个特征词，事迹与特征词一一挂钩。切忌只堆特征词而不落事例，也切忌只复述事例而不提炼词。

2. 答案必含要素（核心）

  • 要素A｜点数≈分值，逐点罗列。 4 分题列四点、5 分题列四点封顶，一点一个"表现+概括"。作答宁多勿漏。
  • 要素B｜"表现（事迹）+ 特征"成对出现。 只叙事不概括、或只给词不给事，均扣分。
  • 要素C｜多维度扫描，避免同质重复。 一个人物要从不同侧面（气节、才识、治绩、待人）取材，不要四点都在说同一件事。
  • 要素D｜心理 / 神态类形象题给"神态词 + 层次递进"。 见范例卡二。

3. 自包含范例卡

【出处】 宝山2026二模·第19题（4分）·《画网巾先生传》（清·戴名世）

【文本背景】 清顺治二年，明亡、清廷强令剃发易服，"不从者死"。一位隐姓埋名的明遗民携二仆匿于山寺，仍着明朝衣冠。被俘后网巾被夺，他便命二仆天天在自己额上"画网巾"，人称"画网巾先生"。面对劝降始终不降，且拒留姓名，最终与二仆从容就义。第①段说他"事尤奇"。

【题目原文】 第①段说"画网巾先生事尤奇"，概括画网巾先生"奇"的表现。（4分）

【官方答案】 ①明亡后仍着明朝衣冠，隐匿山寺，不剃发易服；②被俘后网巾被夺，命二仆在额上画网巾，日以为常，军中呼为"画网巾"；③面对劝降，以"画网巾"为姓名，拒绝投降；④二仆从容就义后，先生笑对死亡，不述姓名，终被戮。

【评分说明】 （4分，每点1分，四点各1分。）

【模板拆解】 - 要素A（点数≈分值）→ 答案恰列 4 点对应 4 分，一点一个"奇"的表现。 - 要素B（表现成对）→ 每点都是具体事迹（着明衣冠/命仆画网巾/以画网巾为名/笑对死亡），不是空泛评价词。 - 要素C（多维度）→ ①外在坚守 ②被俘后的行为 ③拒降 ④赴死，四点分属四个情节阶段，不重复。

【出处】 长宁2026二模·第18题（3分）·《志衍传》（清·吴伟业）

【文本背景】 吴伟业为亡友吴继善（字志衍）作私传。志衍受命赴万里之外的成都为官，明知蜀道已断、蜀事危殆仍慷慨赴任。第③段写他临行前的一段情态："顾置酒张乐，召所与游人人道别，虽握手极笑语，而独坐凝视，椎床弹指，或亲故问之，则浮大白引满，欷歔不复言。"（设酒奏乐，一一与友人道别，握手时谈笑，独坐时却凝视不语、拍床弹指，亲友一问就一饮而尽、叹息不再言语。）

【题目原文】 分析第③段画波浪线部分中志衍的行为表现。（3分）

【官方答案】 志衍在离别之际的行为表现出他故作洒脱背后的不安与悲怆。他与友人握手言笑，看似从容，实则强作镇定；独处时凝视不语、椎床弹指，显露其内心实则焦灼不安；亲友询问时又借酒浇愁，欷歔无言，更体现他内心的悲怆。

【评分说明】 "强作镇定""不安""悲怆"，答出1点得2分，答出2点得3分。

【模板拆解】 - 要素D（神态词+层次）→ 三个采分词 "强作镇定 / 不安 / 悲怆" 层层递进，情绪由表及里。 - 要素B（表现成对）→ 每个神态词都绑定一处动作细节（握手极笑语→强作镇定；独坐凝视椎床弹指→不安；浮大白欷歔→悲怆）。 - 注意计分非线性：1 点 2 分、2 点 3 分——第一点权重最高，务必先答最核心的"故作洒脱/强作镇定"。

二、选材组材 / 整体性分析

1. 考什么 / 思考动作

问"本文选取了哪些材料 / 为何这样组材 / 前后联系如何形成整体感"。思考动作：先列作者选了哪几组事迹，再问"为什么选这些、为什么这样排"——指向同一形象内核的多角度支撑、正面事迹与侧面烘托的搭配、乃至如实记录缺点以求'实录'。

2. 答案必含要素（核心）

  • 要素A｜"多角度选材"要点清。 从才学 / 仪态 / 政见 / 军事等多侧面取材 + 侧面烘托 + 如实记录缺点（体现史传"实录"）。
  • 要素B｜"组材逻辑"要点明映衬 / 铺垫作用。 说清前一组材料如何映衬、铺垫后一组。
  • 要素C｜"整体性 / 前后联系"题按"指认现象 1 分 + 结合文本分析加分"给分，空泛不给分。

3. 自包含范例卡

证据有限： 所读四份原卷（青浦《方琼州传》、宝山《画网巾先生传》、长宁《志衍传》、虹口《范滂传》）的"文言文一"均未单独设"选材组材/整体性"主观题，故本题型暂无自包含范例卡。要素A—C据同类外卷（闵行-2026二模20"前后联系1分+事件分析2分"、杨浦-2026一模17"映衬突出孝友"）归纳，属要点级，作答仍套用"指认现象1分 + 结合文本分析加分"的通用骨架。

三、塑造手法（正侧结合 / 补叙 / 细节 / 侧面烘托）

1. 考什么 / 思考动作

问"某段 / 全文是如何塑造人物的""某处补叙有何作用""某细节 / 侧面描写的作用"。思考动作：先判定手法（正面：语言 / 动作 / 神态 / 外貌；侧面：他人反应、环境衬托、对比烘托；补叙 / 补记），再分析该手法揭示了人物哪几个侧面，最后落到形象特征。

2. 答案必含要素（核心）

  • 要素A｜塑造手法题＝方法（1 分）+ 分析（2 分）+ 形象概括（1 分）。 这是最硬的官方骨架，三块缺一不可。
  • 要素B｜"方法"要正侧兼顾。 既指出正面描写（写了哪些直接事迹），也指出侧面烘托（他人反应）。
  • 要素C｜"分析"要落到具体事例，"形象概括"要给出品格词。

3. 自包含范例卡

【出处】 虹口2025一模·第19题（4分）·《后汉书·范滂传》（节选）

【文本背景】 范滂任功曹，"严整疾恶"。第③段写他两件事：一是把行为违背孝悌仁义者"皆扫迹斥逐，不与共朝"；二是外甥李颂虽是公族子孙却为乡里所弃，中常侍唐衡托太守宗资任用他，范滂"以非其人，寝而不召"（认为不称职就搁置不召）。太守宗资迁怒鞭打书佐朱零，朱零仰头说："范滂清裁，犹以利刃齿腐朽。今日宁受笞死，而滂不可违。"宗资只得作罢。

【题目原文】 第③段是如何塑造范滂这一人物形象的？请加以分析。（4分）

【官方答案】 第③段运用正面描写与侧面烘托相结合的方法塑造范滂的形象。正面描写了他将品行不端之人全都斥退赶走、坚持不任用自己的外甥李颂等事迹；侧面描写了太守宗资对他的格外器重以及书佐朱零宁愿被鞭打也不愿违背他的事。通过正侧结合，表现了范滂大公无私(品行高尚)、执法严明(嫉恶如仇)的形象。

【评分说明】 方法1分，分析2分，形象概括1分。

【模板拆解】 - 要素A（三块骨架）→ 方法 = "正面描写+侧面烘托相结合"（1分）；分析 = 正面举斥逐不端者、不用外甥，侧面举宗资器重、朱零宁笞死不违（2分）；形象概括 = "大公无私、执法严明"（1分）。三块与"1+2+1"精确对应。 - 要素B（正侧兼顾）→ 方法句必须同时点出正面（他自己做的事）与侧面（别人对他的反应），只答一面丢分。 - 要素C（分析落事例）→ 分析部分逐一对应文中事迹，不能只说"运用了侧面烘托"而不举朱零、宗资。

四、评价人物 / 治理能力 / 行为动机

1. 考什么 / 思考动作

问"评价其治理能力""是否堪比古之良吏""为何做出某种选择 / 有何动机""是否认同对某人的评价"。思考动作：先明确表态或定位，再从不同侧面（治理：慎刑断案、爱民兴学、因地制宜、不徇私；动机：从人物自述与处境中分层提炼）分点举证。

2. 答案必含要素（核心）

  • 要素A｜治理能力题＝治绩分维度分点，任一点定额封顶。 见范例卡一（任一点 2 分，满 5 分）。
  • 要素B｜每一维＝"哪件事 + 体现什么能力/品格"，扫描全篇不同官职阶段。
  • 要素C｜动机 / 原因分析题＝从人物自述逐层拆解，一层一点。 见范例卡二。

3. 自包含范例卡

【出处】 青浦2026二模·第19题（5分）·《方琼州传》（马其昶）

【文本背景】 方向（号一菴，世称琼州公）是明代廉吏。任南京户科给事中时，借西安门火灾、雷震孝陵之变上疏弹劾跋扈宦官陈祖生及数十名阿附大臣，屡遭报复下狱、贬为万里之外的多罗驿丞；此后历署三泊县、任资阳尹、治琼州，均赋兴学、公正断案、因地制宜安抚少数民族。作者马其昶篇末赞曰："琼州清节盖世，作吏勤治，尸祝不衰，虽古循良何以过焉？"

【题目原文】 作者称赞方琼州"虽古循良何以过焉"，认为他堪比古时候的良吏，请结合选文内容分析这一观点的依据。（5分）

【官方答案】 刚直敢谏，不惧权贵：任给事中时，借天灾上疏弹劾跋扈宦官陈祖生及数十位阿附大臣，虽遭报复下狱，仍不改其志；勤政爱民，兴学均赋：贬谪多罗后代理三泊县事务，均衡赋税、兴办学校，使当地少数民族归顺接受教化；公正断案，杜绝舞弊：任资阳尹时，将法律条文置于庭中，让囚犯自行翻阅拟定刑罚，使官吏无法作弊，又受委托平反数十件冤案；因地制宜，灵活施政：治理琼州时变刚为柔，以威福晓谕酋长，礼遇耆老，兴办文教，宽恕并和解苻南越兄弟多年诉讼。

【评分说明】 任意一点2分，给满5分为止。

【模板拆解】 - 要素A（分维度、任一点定额）→ 官方列四维（刚直敢谏 / 勤政爱民兴学 / 公正断案 / 因地制宜），任一点 2 分，满 5 分——写足 3 点即到顶，宁可多写一点保底。 - 要素B（"事+能力"成对）→ 每维都是"四字概括 + 具体事迹"格式（如"公正断案，杜绝舞弊：置律于庭…平反数十件"），只写概括词不举事扣分。 - 要素C（扫描各官职阶段）→ 四点分别取自给事中→三泊县→资阳尹→琼州不同任上，覆盖全篇，不重复扎堆。

【出处】 宝山2026二模·第18题（4分）·《画网巾先生传》（清·戴名世）

【文本背景】 承范例卡（画网巾先生）。先生被追问姓名时说："吾忠未能报国，留姓名则辱国；智未能保家，留姓名则辱家；危不即致身，留姓名则辱身……"临刑又说自己"负吾君耳……又以此易节烈名，吾笑夫古今之循例而负义者，故耻不自述也"。题目另附作者赞语："自古守节之士，不肯以姓字落人间者，始于明永乐之世……一夫守义而祸及九族，故多匿迹而死，以全其宗党。"

【题目原文】 阅读下面作者的赞语，结合文本分析画网巾先生不愿意"以姓字落人间"的原因。（4分）

【官方答案】 ①为保全宗族，避免因自己守节而牵连家人；②自认忠未能报国、智未能保家、危不即致身，留姓名会辱及国、家、自身；③嘲笑那些遵循旧例、空有节烈之名而背弃道义的人，耻于自述姓名。

【评分说明】 （4分。）

【模板拆解】 - 要素C（动机分层）→ 三点各是一个动机层次：外部（保全宗族，来自赞语）→ 自我评价（忠智身"三辱"，来自先生自述）→ 价值立场（耻与循例负义者为伍），从他人到自己、从现实到精神层层深入。 - 要素B（每点绑文本）→ ①对应赞语"祸及九族…全其宗党"；②对应"辱国/辱家/辱身"排比；③对应"吾笑夫古今之循例而负义者"。题干明确要求"结合赞语+文本"，两处材料都要用到。

五、评析史传客观性 / 作者身份影响 / 私传阅读态度

1. 考什么 / 思考动作

近两年（尤其 2026）显著新增的"元评价"题型：问"作者身份对文章客观性的影响""对古代私传应持怎样态度""某句史评是否得当"。思考动作：先明确史传/实录应有的"客观—褒贬并存—征实"标准，再指出该文因作者身份/私传性质可能造成的偏向（有褒无贬、失真风险），最后给辩证态度（既承认独特价值又警惕局限）。

2. 答案必含要素（核心）

  • 要素A｜态度必须辩证：不盲从尽信、也不全盘否定（1 分）。 单边表态直接压分。
  • 要素B｜"失真风险"一侧：指出私传/身份带来的情感倾向与失实可能，并举文中/补充材料证据。
  • 要素C｜"独特价值"一侧：指出私传保留鲜活细节、呈现人物复杂性、可补正史之阙的价值，并举文中证据。
  • 两侧证据均衡给分（各占定额）。

3. 自包含范例卡

【出处】 长宁2026二模·第19题（4分）·《志衍传》（清·吴伟业）

【文本背景】 吴伟业为亡友吴继善（字志衍）作私传，文中写志衍营救恩师张溥、登黄山抒怀、临行成都时的愁楚、贻书悲叹，最后"骂不绝口"殉难、一门四十余人同日遇害。题目附补充材料：《钦定胜朝殉节诸臣录》指出"吴继善降贼被杀，而吴伟业作传，乃称其大骂捐躯，更见私集之阿谀难据"——即质疑私传因私人关系而美化传主、不足为信。

【题目原文】 小宁读了下面的补充材料后，对古代私传的阅读价值表示怀疑。你认为读者对古代私传应持怎样的阅读态度？请结合补充材料并紧扣本文内容，向小宁阐述你的看法。（4分）

【官方答案】 小宁你好。阅读古代私传，既不能盲从尽信，也不宜全盘否定。私传易带情感倾向，存在失实风险。如材料指出吴继善实为降贼被杀，吴伟业却因与他有亲密的私人关系而将其塑造成骂贼殉节的忠臣，可见私传不可直接等同于信史，需结合多方史料互证。同时，私传往往保留了传主生活中的鲜活细节，如吴继善营救恩师、登山抒怀、临行愁楚、贻书悲叹等，呈现了人物的复杂性，可补正史之阙，具备独特的参考价值与文学审美价值，不应一概抹杀。

【评分说明】 不应盲从尽信或全盘否定，1分；分析失真风险与独特价值，各2分。给满4分为止。

【模板拆解】 - 要素A（辩证表态）→ 开篇"既不能盲从尽信，也不宜全盘否定"= 1 分。这是入场券，缺则丢分且方向错。 - 要素B（失真风险）→ 借补充材料举证"吴继善实为降贼被杀，作者因私人关系塑造成骂贼殉节的忠臣→需多方史料互证"= 2 分。必须用到补充材料。 - 要素C（独特价值）→ 举本文细节"营救恩师、登山抒怀、临行愁楚、贻书悲叹→呈现人物复杂性、补正史之阙、有文学审美价值"= 2 分。必须回扣本文情节。 - 结构提示：题干设定了"向小宁阐述"的对话情境，答案以称呼"小宁你好"开头，注意应答口吻。

六、句读断句（技能型·采分模板）

1. 考什么 / 思考动作

在无标点的文言语段中断句，通常 3—5 处。思考动作：抓虚词（者/也/矣/乎/焉/而/则）、抓对话标志（曰/云）、抓句式对称与主谓完整、抓人名官职专名不可断开。

2. 答案必含要素（核心·采分模板）

  • 计分方式＝"错一处扣一分，扣到 0 分为止"。 多断、漏断、错断均按处扣分——只在有把握处下笔，切忌为凑数硬断。
  • 作答口诀：①先通读定大意、锁定人物与动作主语；②每个完整"主+谓（+宾）"后优先考虑断；③句末虚词后断、句首虚词（夫/盖/故/是以/岂）前断；④"曰/对曰"后断；⑤人名、官名、地名连读不断。

3. 自包含范例卡

【出处】 青浦2026二模·第17题（5分）·《方琼州传》

【文本背景】 方向上疏弹劾宦官陈祖生及阿附大臣，疏文中一段（需断五处）： 原句： 祖生擅作威福渔猎军民而诸臣持禄阿附无忠直节乞赐黜罚回天变。

【题目原文】 第②段画线部分有五处需加句读，请用"/"把这五处标识出来。（5分）

【官方答案（断法）】 祖生擅作威福／渔猎军民／而诸臣持禄阿附／无忠直节／乞赐黜罚／回天变。

【评分说明】 错一处扣一分，共5分。

【模板拆解】 - "祖生擅作威福 / 渔猎军民" → 两个并列谓语短语（作威福｜渔猎军民）各自完整，之间断。 - "而诸臣持禄阿附" → 句首虚词"而"前断，转向新主语"诸臣"。 - "无忠直节 / 乞赐黜罚 / 回天变" → 三个动作短语（无…节｜乞赐…｜回天变）各自成句，逐一断开。 - 采分：错一处扣一分——五处全对得 5 分，断错或漏断一处减 1 分。

【出处】 宝山2026二模·第16题（4分）·《画网巾先生传》

【文本背景】 先生失去网巾后，对二仆交代要在额上画网巾的一段话（需断四处）： 原句： 今吾遭国破即死岂可忘祖制乎汝曹取笔墨来为我画网巾额上。

【题目原文】 第②段画线部分有四处需加句读，请用"/"把这四处标识出来。（4分）

【官方答案（断法）】 今吾遭国破／即死／岂可忘祖制乎／汝曹取笔墨来／为我画网巾额上。

【评分说明】 （4分，错一处扣一分。）

【模板拆解】 - "今吾遭国破 / 即死" → 主谓完整（吾遭国破）后断，"即死"另起。 - "岂可忘祖制乎" → 反问句，句首"岂"前已断、句末虚词"乎"后断（此处"乎"后接新主语"汝曹"）。 - "汝曹取笔墨来 / 为我画网巾额上" → 换主语"汝曹"起一句，其后祈使句再断。 - 口诀命中点：反问标志"岂…乎"、换主语"汝曹"、句末虚词"乎"，都是高把握断点。

七、翻译（技能型·采分模板）

1. 考什么 / 思考动作

翻译传记文中画线句（多 4—7 分）。思考动作：逐字对译、圈定关键实词与特殊句式（判断/被动/宾语前置/省略），补出省略主语，落实每个采分点。

2. 答案必含要素（核心·采分模板）

  • 计分方式＝"关键实词 / 每分句为采分点，一点 1 分（难句可 2 分）"。 漏译或错译一个采分词即扣一分。
  • 作答步骤：①切分为若干分句/短语；②每个分句锁定 1 个关键实词或句式作为踩分锚点；③补省略主语（用括号）；④直译为主，句式标志（被动"为…所"、判断"…者…也"、宾语前置）须译出；⑤检查每个采分点是否落地，人名官名照抄不译。

3. 自包含范例卡

【出处】 长宁2026二模·第17题（4分）·《志衍传》

【文本背景】 宾客劝志衍暂缓赴危险的成都赴任，志衍表明恪守臣职、义无反顾之志： 原句： 吾既受成命矣，人臣守官，其敢以利害辞？且今日何乐土之有？

【题目原文】 把第③段画横线部分译成现代汉语。（4分）

【官方答案（译文）】 我已经接受了已发布的命令，作为臣子恪守官职，难道敢因为利害得失而推辞？况且如今哪里有什么安乐的地方呢？

【评分说明】 "既受成命矣""人臣守官""其""辞""何乐土之有(宾语前置)"，错一处扣1分，扣到0分为止。

【模板拆解】 - "既受成命矣" → 采分点："既"=已经、"成命"=已发布的命令。 - "人臣守官" → 采分点：补出/译准"作为臣子恪守官职"。 - "其敢…辞" → 采分点："其"=难道（反问语气词）、"辞"=推辞，两个词各是一处。 - "何乐土之有" → 采分点：宾语前置句式，须还原为"有何乐土"译出"哪里有安乐之地"。 - 五个采分点对应五处扣分，特殊句式（宾语前置）必须译出结构才得分。

【出处】 虹口2025一模·第18题（7分）·《范滂传》

【文本背景】 范滂投版弃官而去，郭林宗（郭泰）为此责备光禄勋陈蕃： 原句： 郭林宗闻而让蕃曰："若范孟博者，岂宜以公礼格之？今成其去就之名，得无自取不优之议也？"

【题目原文】 把第①段画线句译成现代文。（7分）

【官方答案（译文）】 郭泰(林宗)听说后就责备陈蕃说："像范滂(孟博)这样的人，怎么能够用一般官府的礼仪来拘束他（或：让他遵守一般官府的礼仪来拜见你）呢？现在你成全了他宁愿辞官也不肯屈就的美名，岂不是自己招来了不懂得优待贤士的恶评（或：不好的评论）吗？"

【评分说明】 前三个分句各1分，后两个分句各2分。

【模板拆解】 - "郭林宗闻而让蕃曰" → 分句1（1分）："让"=责备（关键实词，勿误作"谦让"），人名"郭林宗/孟博"照抄。 - "若范孟博者" → 分句2（1分）："若…者"=像…这样的人。 - "岂宜以公礼格之" → 分句3（1分）："岂宜"=怎么能够、"格"=拘束/约束。 - "今成其去就之名" → 分句4（2分）："成"=成全、"去就之名"=宁愿辞官不屈就的美名，难句权重加倍。 - "得无自取不优之议也" → 分句5（2分）："得无…也"=岂不是…吗、"不优之议"=不懂优待贤士的恶评，双重难点计 2 分。 - 规律：长句按分句差异化赋分，越靠后、句式越难的分句权重越高，务必逐句抠关键词。

八、词义解释（技能型·采分模板）

1. 考什么 / 思考动作

解释加点实词、或为加点词选择正确义项（通常各 1 分）。思考动作：据传记语境定一词多义、注意词类活用与古今异义，用现代双音词精准替换。

2. 答案必含要素（核心·采分模板）

  • 计分方式＝逐词给分、格式化，每词 1 分。 评分只认精准命中的核心义，泛化表述零分。
  • 作答模板：①代入语境判断词性（动/名/形），先定活用；②在常见义项中选贴合上下文动作逻辑的一个；③用一个精准现代词作答，不写句子、不作解释性长句；④警惕古今异义与偏义复词。

3. 自包含范例卡

【出处】 虹口2025一模·第15、16题（各2分）·《范滂传》

【文本背景】 - 第①段："滂执公仪诣蕃"——范滂持下属礼节去见光禄勋陈蕃。 - 第①段："复为太尉黄琼所辟"——又被太尉黄琼征召。 - 第③段："请署功曹"——太守请范滂充任功曹。 - 第③段："寝而不召"——范滂搁置（对外甥李颂的任用请求）而不征召。

【题目原文】 15．写出下列加粗词在句中的意思。（2分）(1)滂执公仪诣蕃　(2)复为太尉黄琼所辟 16．为下列句中加粗词选择释义正确的一项。（2分）(1)请署功曹（ ）A.布署 B.考核 C.充任 D.记录　(2)寝而不召（ ）A.搁置 B.休息 C.罢免 D.平息

【官方答案】 15．(1)诣＝拜访　(2)辟＝征召。　16．(1)署＝C（充任）　(2)寝＝A（搁置）。

【评分说明】 各题2分，逐词1分。

【模板拆解】 - "诣"→ 拜访（据"执公仪…蕃"的动作逻辑：持礼去见，故取"到…去、拜访"义，非"造诣"）。 - "辟"→ 征召（"为…所辟"被动句，上级征召下级为官的专义）。 - "署"→ 充任（"请署功曹"=请他担任功曹，官职语境；干扰项"布署/记录"皆偏离）。 - "寝"→ 搁置（"寝而不召"=把请求压下不办，非"睡觉"本义；这是一词多义的语境活用）。 - 规律：逐词精准命中核心义，一个现代双音词作答即可；选择题干扰项多为该词的其他常见义或形近义，靠语境动作逻辑排除。
