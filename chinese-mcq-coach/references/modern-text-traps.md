<overview>
现代文阅读（论述类、实用类、文学类文本）选择题的干扰项，几乎全部落在以下三大类、十余种套路之中。每种给出【特征】【识别方法/破绽词】【例子】。做题时对照指认，出题人的花招翻来覆去就这几板斧。
</overview>

<category name="第一大类：逻辑谬误型">
不改动原文信息，而在逻辑关系上做手脚。

<trap name="1. 强加因果 / 颠倒因果">
- **特征**：原文中两件事只是先后发生或仅相关，选项硬说成因果；或直接因果倒置。
- **破绽词**：看到"因为…所以…""导致""促使""使得""从而"，立刻回原文核查因果链是否成立。
- **例子**：原文"他心情不好，碰巧丢了钱包"；选项"他因为丢了钱包而心情不好"（时间/因果颠倒）。
</trap>

<trap name="2. 偷换概念 / 张冠李戴">
- **特征**：把甲事物的特点、作用悄悄安到乙事物头上；或用近义词替换但内涵已变。
- **破绽词**：核对选项的主语和宾语，回原文确认它们是不是"一家子"。如"文化自信"换成"文化自大"、"主要因素"换成"根本动力"。
- **例子**：原文"诗歌的韵律美"；选项"诗歌的格式美"（"韵律"≠"格式"）。
</trap>

<trap name="3. 以偏概全 / 以全否偏">
- **特征**：把部分说成整体、个别说成普遍；或反过来用一个反例否定全部。
- **破绽词**：紧盯范围词——"全都""无一例外""只是""仅仅""之一""所有""任何"。
- **例子**：原文"很多学生喜欢阅读"；选项"学生都喜欢阅读"（"很多"被改成"都"）。
</trap>

<trap name="4. 绝对化判断">
- **特征**：把原文或然、相对的表述，换成必然、绝对的表述。这种最"送分"。
- **破绽词**："一定""必然""绝对""完全""彻底""无疑""只要…就"。
- **例子**：原文"这种方法可能有效"；选项"这种方法必然有效"。
</trap>

<trap name="5. 结论无据 / 过度推断">
- **特征**：原文没有依据或依据不足，选项硬推出一个看似深刻的结论。
- **破绽词**：自问"这个观点作者在文中明确说过吗？哪怕暗示过吗？"原文无据，再正确的名言也是错项。
- **例子**：原文讲环保重要；选项"因此应废除所有工业"（原文根本没提废除工业）。
</trap>
</category>

<category name="第二大类：信息整合型">
在整合原文多处信息时，故意扭曲、遗漏或添加。

<trap name="6. 未然变已然 / 或然变必然">
- **特征**：把设想中、还没发生的事说成已发生；把可能的事说成确定的事。
- **破绽词**：死抠时态语气——"将""有望""预计""拟"vs"已经""实现了""成为"。一字之差，生死之别。
- **例子**：原文"这项技术有望明年上市"；选项"这项技术已经上市"。
</trap>

<trap name="7. 曲解文意">
- **特征**：不篡改文字，而把褒义说成贬义、把批评说成赞扬，搞错情感基调。
- **破绽词**：读懂作者的用词和态度，看字面更看态度。
- **例子**：原文作者明显在讽刺某现象；选项"作者对这种做法表示赞赏"。
</trap>

<trap name="8. 信息杂糅 / 拼凑嫁接">
- **特征**：把文中几个不相关的信息点强行拼在一起，构成看似整合、实则荒谬的新信息。
- **破绽词**：把选项信息点拆开，逐一回原文定位，看它们在原文是不是"一家人"、能不能这样组合。
- **例子**：A段讲苹果营养，B段讲香蕉口感；选项"苹果因其口感而受喜爱"（把香蕉的特点嫁接给苹果）。
</trap>

<trap name="9. 信息遗漏 / 概括失当">
- **特征**：概括原文时故意漏掉关键限制条件、适用范围或重要因素，使概括片面或错误。
- **破绽词**：对整段的概括，检查是否保留了所有核心要素，尤其注意定语、状语等限定成分。
- **例子**：原文"在资金有限的情况下，这个方法很实用"；选项"这个方法很实用"（漏了关键前提）。
</trap>
</category>

<category name="第三大类：程度与范围型">
最精细、最需要文本比对功夫的一类。

<trap name="10. 改变性质 / 程度轻重">
- **特征**：把批评改成否定、怀疑改成颠覆、"初步成果"夸大为"历史性突破"。
- **破绽词**：对程度极敏感——"影响"vs"决定"、"参与"vs"主导"、"有助于"vs"取决于"。
- **例子**：原文"他的研究对该领域有影响"；选项"他的研究决定了该领域的发展"。
</trap>

<trap name="11. 扩大 / 缩小时间或空间范围">
- **特征**：把某时代、某地域的事推广到所有时代、所有地域（或相反缩小）。
- **破绽词**：见到"自古以来""全球各地""在西方""在古代""所有时期"等时空限定，高度警觉原文有无此限定。
- **例子**：原文论述"唐宋时期的文人"；选项"中国古代文人皆是如此"。
</trap>
</category>

<ultimate_method>
**终极心法**
- 唯一武器：**文本比对**——拿选项每一个分句回原文找一模一样的证据。
- 核心态度：**六亲不认**——不联想、不推断、不"我觉得"、不"应该是"，一切以白纸黑字为准。
- 刷题练法：每做一道题，就尝试指认干扰项属于以上哪一类。很快就会发现出题人的花招就这几板斧。
</ultimate_method>
