<overview>
古诗文选择题分两大块：文言文阅读的"内容理解/概括分析"选择题，和诗歌鉴赏的"理解与赏析"选择题。除现代文那套通用陷阱（强加因果、以偏概全、曲解文意等同样适用）外，古诗文还有一批专属陷阱。做题原则同样是回原文（原诗/原文）逐句比对。
</overview>

<category name="第一大类：文言文'概括分析题'专属陷阱">
题干多为"下列对原文有关内容的概括和分析，不正确的一项是"。

<trap name="1. 事件张冠李戴 / 时间地点错位">
- **特征**：把甲做的事安到乙头上；或把不同时间、不同地点发生的事混为一谈、颠倒先后。
- **破绽词**：核对选项中的人名、官职、时间、地点，逐一回原文定位。命题人最爱把"甲劝谏"说成"乙劝谏"。
- **例子**：原文是"太守上表",选项说成"县令上表"。
</trap>

<trap name="2. 曲解词义 / 以今律古">
- **特征**：把文言实词、虚词按现代义理解，或曲解一词多义、古今异义词，导致对情节判断错误。
- **破绽词**：警惕"妻子（妻子儿女）""地方（土地方圆）""亲戚（族内外亲属）""行李（外交使者）"等古今异义；判定分析题时先核实关键词的文言义。
- **例子**：把"素善留侯张良"的"善"（交好）理解成"善良"。
</trap>

<trap name="3. 无中生有 / 强加因果">
- **特征**：选项增添原文没有的情节、评价或因果关系，看似合理实则文中无据。
- **破绽词**：选项中表因果、表目的的连词（"因为""为了""于是"）要回原文核查；找不到对应文字即为编造。
- **例子**：原文只叙述某人辞官，选项添"因不满朝政腐败而辞官"，但原文未提原因。
</trap>

<trap name="4. 拔高/贬低人物 / 曲解评价">
- **特征**：夸大或贬低传主的品行、功绩、动机，或把作者的中性叙述解读成明确褒贬。
- **破绽词**：注意"体现了…高尚品质""意在批判"这类拔高性表述，回原文看是否有支撑。
- **例子**：原文只写某官"减免赋税"，选项拔高为"一生清廉、深受百姓爱戴"。
</trap>

<trap name="5. 归纳以偏概全 / 程度失当">
- **特征**：用一个事例概括为一贯品行；或把"曾经""偶尔"说成"总是""一向"。
- **破绽词**：范围词与频率词（"始终""历来""每每""凡"）回原文核查。
</trap>
</category>

<category name="第二大类：诗歌鉴赏选择题专属陷阱">
题干多为"下列对这首诗（词）的理解和赏析，不正确的一项是"。

<trap name="6. 手法误判">
- **特征**：把修辞/表现手法张冠李戴——如把"比喻"说成"拟人"、"借景抒情"说成"直抒胸臆"、"用典"说错典故、"对偶"误作"排比"、"动静结合"张冠李戴。
- **破绽词**：凡选项断言某种手法，回诗句验证是否真用了该手法。手法术语是重灾区。
- **例子**：选项称"运用了拟人手法"，但诗句实为比喻。
</trap>

<trap name="7. 意象 / 意境情感误读">
- **特征**：把冷寂的意境说成明丽，把哀愁误读为闲适；或把典型意象的固定内涵搞错（如"折柳"送别、"杜鹃"悲苦、"鸿雁"思乡）。
- **破绽词**：抓关键意象和情感基调词，比对全诗整体情感是否一致。注意"乐景写哀"等反衬。
- **例子**：一首明显的羁旅思乡诗，选项说"表达了归隐田园的闲适自得"。
</trap>

<trap name="8. 词句理解望文生义">
- **特征**：脱离语境曲解某字某句，或把炼字的表达效果说错。
- **破绽词**：对"诗眼"、关键动词/形容词的赏析，回到具体诗句和上下文验证。
</trap>

<trap name="9. 内容层次 / 结构错位">
- **特征**：把上片说成下片的内容、把写景句说成叙事句、颠倒诗歌的时空或逻辑层次。
- **破绽词**：核对选项指涉的是哪一联/哪一句，与原诗对应位置比对。
</trap>

<trap name="10. 无中生有 / 过度引申">
- **特征**：给诗句强加原文并无的主旨、背景或象征意义（"表达了诗人对国家命运的忧虑"等宏大拔高）。
- **破绽词**：宏大主旨类断言尤其可疑，需诗中有明确依据方可成立。
</trap>
</category>

<ultimate_method>
**古诗文破解要点**
- 通用陷阱（因果、绝对化、以偏概全、曲解文意）在古诗文同样适用，先扫一遍。
- 文言题：先落实关键词的**文言义**，再核对人物—事件—时间—地点的对应。
- 诗歌题：手法术语和情感基调是两大重灾区，凡断言必回诗句验证。
- 一切以原文/原诗为准，不用现代常识脑补古人。
</ultimate_method>
